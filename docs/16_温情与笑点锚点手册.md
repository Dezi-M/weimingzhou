# 《未名舟》温情与笑点锚点手册

本手册只管一件事：让船上有人味。

它不是“每章必须搞笑”的规则。它更像一只小抽屉：当最近几章太紧、太疼、太硬，或者五十章循环需要情绪缓冲时，从这里拿一点烟火气，让读者知道人还在，船还在，感情还在。

## 使用口径

- 温情和笑点不是水章，它们是后面大战会疼的资产。
- 不要在真正死亡、背叛、失声的正中间强行插笑点；可以放在伤口之前、余波之后，或两个压力之间。
- 笑点优先来自人物的错、穷、嘴硬、职业习惯和旧物尴尬，不来自段子。
- 一次只放一点。船上有人互损一句、分粥时少一勺、旧锅又漏一点，往往比整段抒情更好听。
- 幽默不负责冲淡痛苦，只负责证明人还没被痛苦磨平。笑完以后，伤口、账、敌人的钩子还在。

## 固定温情/喜剧组合

| 组合 | 手感 | 可用场景 |
|---|---|---|
| 老周嘴硬 vs 陆小满扎心 | 老周把心疼说成嫌弃，陆小满把他的嘴硬记成账。 | 分粥、补锅、战后清点、孩子不肯吃饭。 |
| 铁牙讲冷话没人接 | 他从“线”“站”“疼”说起，别人以为他在骂人，其实是在护边界。 | 守夜、绑索、断臂疼、船骨回应。 |
| 孙福来钥匙乱响 | 他总想把东西藏好，结果钥匙比他先露怯。 | 米柜、库门、合法港诱惑、旧锁辨伪。 |
| 许青刀母子互相拆台 | 母亲不把牺牲说漂亮，儿子也不把勇敢说满。 | 守槽、分饼、受伤、别人夸许青刀时。 |
| 梁春枝断秤损人 | 她拿市井话压住大词，骂人时顺手救人。 | 称粮、称伤、称债、敌人文书太漂亮时。 |
| 卿禾被照看 | 她灯耗后不必大哭，可以有人递旧布、遮风、让她先坐下。 | 高压章后半页、夜里修灯、孩子围着灯认错字。 |

## 幽默风趣触发器

《未名舟》的幽默不要像段子手进舱，更像船板漏风时有人骂了一句，大家忽然能喘气。

优先用这五类：

- 破话：老周、梁春枝把大道理拆成难听但准的话。
- 错话：孩子、半饼大孩子、刚夺回声音的人说错半个字，旁人不纠正，只顺手接住。
- 小算盘：邱成怕裂碗、孙福来怕丢钥匙、老周怕少盐，越算越暴露人味。
- 职业习惯：苏晚把玩笑落到伤口，铁牙把玩笑说成军令，陆小满把嘴硬记进账。
- 旧物尴尬：锅盖补完又漏，钥匙藏不住响，断秤砣砸歪，木牌字写得丑但有用。

不适合用：

- 敌人刚杀人、封口、夺亲属称谓的正中间。
- 角色刚做出不可逆牺牲的第一口气。
- 需要读者恨敌人的场面里，别把刀口逗软。

适合用：

- 高压判词落下前，让普通人先说一句没出息但真的怕。
- 小胜之后，让某件旧物坏得很没面子。
- 战后清点时，让角色为了半勺盐、一截绷带、一块碗瓷吵两句。
- 新脑洞首次登场时，让人先误用、错认，笑一下再疼一下。

## 日常资产池

这些东西不必参与战斗，主要参与“活着”：

- 半截铅笔头。
- 缝了一半的旧布边。
- 碎碗拼的小花盆。
- 补过三次的锅铲。
- 缺齿钥匙串。
- 写错字的木牌。
- 一小包盐。
- 旧鞋带。
- 洗不净的碗底灰。
- 门线结。
- 漏风的袖口。
- 孩子藏起来的一粒米。

## 056-100 可用小场面

- 半枚白钉还在卿宸掌心里冷，老周说“拔钉另算工钱”，苏晚抬头看他一眼，他立刻改口“算我欠的”。
- 五枚问处发热，邱成第一反应不是高兴，是把缺口碗往怀里捂：“它要再裂，谁赔？”梁春枝说“赔你半个好听的哭声？”
- 纸雨第一次落下，孩子掌心多了临时用途名，吓得把手塞进老周锅烟里，老周骂“你当锅是洗手盆？”
- 孙福来想悄悄藏起一枚白票残角，钥匙串先响，全船一起看他，他自己先咳一声：“我试试它响不响。”
- 卿禾校牌忽然变凉，她说“不认路”，老周顺口问“那认饭点吗”，没人笑太久，因为校牌下一瞬更凉。
- 梁春枝称饭时把半粒盐也算上，老周气得说“你再称下去，锅烟都得交税。”

## 软节奏

- 最近 8-10 章如果全是外压、追杀、规则反制，可以顺手放一个温情或笑点锚点。
- 每 50 章商业循环里，尽量有一个读者能复述的纯温情或纯笑点小场面。
- 若当前章是大战余波、关系变质、救人后喘息，可以不掉新道具，只让旧物多一道新伤痕，或让一个人说一句不漂亮但很真的话。
