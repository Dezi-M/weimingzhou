# 039 开写准备：猎足踩影

本文件只服务 039 开写，负责把已发 001-038 的连续性锚点和 `06_039后新素材库_人间开阵.md` 的新战场素材压成可落笔的单章驾驶舱。039 完成后，把真实发生的战果、代价、敌方样本和章末钩回写到 `14_当前故事状态账本.md`，本文件不作为长期总纲。

## 输入复核

- 章号：039
- 暂定章名：第三十九章 猎足踩影
- 上章钩子：038 章末，断桥白板翻出密密麻麻的脚印；无形脚印一步一步朝未名舟走来。
- 本章主发动机：不是抛掉前文重开，也不是继续验桥查票，而是把 038 裂出的脚印升维为猎足，让读者第一次看见“高位面不先杀人，先踩人和旧物之间的关系”。
- 本章一句话：第三个脚印落下时，甲板没碎，七十个人的影子先矮了一寸。
- 本章结果：未名舟看清第一只猎足“不踩肉，踩关系”；卿宸挡下第一息；孙福来、邱成、卿禾等人各有微小主动动作；章末校牌影子被踩出灯芯轮廓，逼出 040 的锅灰描足。

## 1-38 承接原则

039 不能像另起一本书。猎足踩影要踩在读者已经认识的旧关系上：

| 前文范围 | 已成立的读者记忆 | 039 接法 |
|---|---|---|
| 001-005 | 卿禾不是燃灯童；无名船骨醒；归家灯和半截“禾”字 | 校牌影子被踩出灯芯轮廓时，读者要想起卿禾曾被写成灯，也想起她不是灯 |
| 006-010 | 七十张嘴、第一顿非法粥、童灯队、半声“到”、船头木牌 | 猎足让人影变矮时，七十人不是背景；他们曾经一个个按灰印留下，现在一个个被压成用途 |
| 011-013 | 苏晚、药箱、急救方舱、救命债不能删 | 本章不主打医线，但苏晚要能看出“这不是伤，却比伤更难包”，为 051 水药床战场埋口 |
| 014-020 | 铁牙、兵骨、守线、压舱者、黑航道第一战 | 猎足不能只像玄学压迫，铁牙和守线者要本能站位，发现刀索挡不住影子 |
| 021-030 | 老街上船、旧绿门、缺口碗、断秤、旧钥、第一规 | 039 的旧物不是新道具；钥、碗、秤、门都已经有生活来处，猎足是来拆这些来处 |
| 031-034 | 第一道回应火、追声救援、半个我、旧校牌敲下课 | 039 的校牌冷光要接“敲下课”和“照回人”的旧感，不要只当灯光特效 |
| 035-038 | 不亮灯残舟、守住三丈、反剪补票钳、踩歪验票桥 | 039 从“桥孔里来脚印”顺接，三丈线和残舟帆影必须在场，证明这不是换敌人，而是敌压升维 |

本章的“新”只在敌人层级，不在情感基础。读者应该感觉：原来前面那些锅、碗、钥匙、校牌、木牌、残舟和三丈线，全都被敌人看见了。

## 必接旧物 / 旧人清单

- 船头木牌：接 010 的半声“到”、023 的求救木牌、030 的第一规。039 用它挡第一息，不让它变成万能判词。
- 卿禾旧校牌：接燃灯童、归家灯、旧校牌、敲下课。039 只照出灯芯轮廓，不提前支付 046 的记忆大债。
- 老周铁锅：接第一顿非法粥、饭香救人、回应火。039 只让焦味被压低或锅灰微动，真正描足留给 040。
- 邱成缺口碗：接怕死也留下、两瓶水分不下、碗扣灰印。039 不直接倒扣成阵眼，只先拒绝接黑水。
- 孙福来旧钥串：接旧绿门、后巷、半声藏路、钥匙先露怯。039 的高光是“不开门”，不是炫技开锁。
- 梁春枝断秤：接老街公平、小账、压白线。039 可让她先意识到影子有重量，但正式称影留给 044。
- 铁牙焦索 / 守线：接兵骨和守线，不需要抢戏；他第一个发现实体防线挡不住影压。
- 残舟帆影：接不亮灯残舟和三丈线；039 只能帆影受压，不能并船、不能归顺。

## 第一屏

建议第一句：

> 第三个脚印落下的时候，未名舟没有晃。先矮下去的是人的影子。

第一屏要完成三件事：

1. 接住 038 的“脚印一步一步朝未名舟走来”，不跳场。
2. 马上给新鲜恐怖：脚印不踩甲板，不踩肉，先踩影子。
3. 不解释“终相猎场”，只写身体感：人突然觉得自己只剩一种用途。

可写的第一屏动作：

- 旧钥串先响，孙福来还没动，钥影已经朝旧绿门弯过去。
- 邱成碗里没有水，碗影却像被灌成配给碗。
- 卿禾校牌没亮，影子先冷成一截细灯芯。
- 残舟帆影在三丈线外往下沉，船体却没有动。

## 八格驾驶舱

1. 危险：
   - 第一只猎足从断桥白板的票孔里走出。
   - 它不攻击船体，先踩残舟帆影、旧钥影、缺口碗影、旧校牌影。
   - 每踩一次，船上对应的人都会短暂觉得自己“只能做一种事”。

2. 目标：
   - 在猎足踩进三丈线前，看清它到底踩什么。
   - 保住至少一个旧物和人的关系不被拖走。
   - 不收断桥白板，不让敌方入口上船。

3. 敌压：
   - 猎足第一轮打法是“踩影”。
   - 它故意不砸旧物，而是让旧物看起来像自己想被重新定义：钥匙想开门，碗想接黑水，校牌想照编号，帆影想并船。
   - 它等未名舟误判，只要众人急着抢物，就能采到“谁最在乎哪件旧物”的关系样本。

4. 行动：
   - 卿宸把船头木牌横进脚印和钥影之间，掌心被压出一枚短脚印，替众人争半息。
   - 孙福来按住钥串，不开门，反而让钥齿咬住自己掌心，说这串钥匙不认脚，只认人。
   - 邱成明明渴，仍把碗口偏开，不让碗影接黑水；他还没倒扣成阵眼，只先守住“这碗不是配给口”。
   - 卿禾抬校牌照向人，不照向脚印；校牌没有成功照回，只照出一个被压成灯芯的轮廓。
   - 残舟不越三丈线，只让帆影挣了一下，证明它也在被踩。

5. 战利品：
   - 明确情报：猎足不踩肉，踩关系。
   - 小战法种子：木牌可暂时隔断“脚印和旧物影子”的落点。
   - 敌方样本的影子：脚印压过后，甲板钉缝里会留下极薄的黑边，040 可由锅灰描出来。

6. 损失/代价：
   - 卿宸掌心多一枚短脚印压痕，旧伤慢半拍。
   - 孙福来钥齿割掌，钥声被猎足记住一声。
   - 邱成碗底白痕加深，碗里明明空着，却像少了一口水。
   - 卿禾校牌霜冷一寸，暂时没丢记忆，但读者知道这条线会疼。
   - 残舟帆影沉半寸，不并船也要付战场代价。

7. 感官：
   - 黑水没有水花，只有影子被压薄的纸声。
   - 钥匙响得太轻，像有人在门外试锁。
   - 空碗里有水声，但碗壁是干的。
   - 校牌冷得贴手，冷意像细灯芯。
   - 锅盖旧焦味被脚印压低，像火被人用脚踩灭。

8. 关系：
   - 卿宸不是替大家开挂，而是挨第一下，把选择权递回去。
   - 孙福来从“钥匙先露怯”的笑点，转成第一个守住旧物关系的人。
   - 邱成继续心疼碗，但这次心疼不是笑点，是他不让碗变成敌方配给口。
   - 残舟仍不谢、不并、不归顺；只用帆影证明它也在这场战里。

9. 宏大线：
   - 七奇锋只露种子：开阙来自钥，悬衡来自碗，返灯来自校牌。
   - 六大战军不封军，只埋赤釜、晨灯、守岸的早期器官。
   - 终相猎场第一次露面，不解释全宇宙，只让读者记住“它把万物压成唯一用途”。

## 意外钩

- 获得什么：不是实体战利品，而是第一条敌招情报：“它踩关系。”
- 表面用处：知道脚印不是普通追兵，可以暂时防错。
- 真正用处：为 040 锅灰描足、042 三丈线伏击、050 人间开阵打地基。
- 代价：敌人也采到己方关系样本，尤其是钥声、校牌冷光、碗底白痕和残舟帆影。
- 旧账折痕：孙福来的钥声后续会被敌人仿；卿禾校牌终会付记忆债；邱成的碗会从“怕漏水”升级到“敢扣影”。

## 情绪钩

余笑未散 -> 影子先矮 -> 普通人守物 -> 看清敌招 -> 校牌影成灯芯，心口收紧

本章不求大胜，要让读者从“脚印来了”变成“这东西太阴，它不是杀你，是把你和你活过的日子拆开”。

## 名场面候选

一句话复述：

> 猎足踩下来的时候，未名舟没碎，人的影子先矮了；卿宸用掌心替七十个人挡住半息，孙福来第一次没开门，而是让钥匙咬住自己。

可用短句：

- “它不踩船。”
- “它踩的是你为什么还护着这东西。”
- “钥匙不认脚，只认人。”
- “我的碗不接它的水。”
- “别急着赢，先别让它知道你们疼在哪。”

## 软规则预检

- 规则先疼再命名：通过影子变矮、钥影弯门、碗里空水声、校牌冷光来表现，不开设定课。
- 人先动：孙福来按钥、邱成偏碗、卿禾照人、卿宸挡第一息。
- 旧物不自动显灵：钥、碗、校牌、帆影都只是回应人的动作。
- 敌人智商在线：猎足不急着杀，先采关系样本，为后续拴影和换影做铺垫。
- 章末有实体钩：校牌影子被踩出灯芯轮廓，锅灰可在下一章开始描足。

## 开写提醒

1. 不要在 039 解释“终相 / 复相 / 万象开阵”。
2. 不要让卿宸喊“人间，开阵”。
3. 不要让旧物直接变武器。
4. 不要让残舟越线或归顺。
5. 不要让本章胜得太满。本章最好的爽感是“终于看懂敌人第一刀”，最好的痛感是“敌人也看懂了我们疼在哪里”。

如果正文写得慢，可以把中段压成三次脚印：

- 第一脚：钥影弯门，孙福来守钥。
- 第二脚：碗影接水，邱成偏碗。
- 第三脚：校牌影成灯芯，卿宸挡半息，卿禾照人失败。

最后一拍落在校牌影子：

> 校牌没有亮。
>
> 它的影子亮了。
>
> 那影子里，有一根很细、很细的灯芯。
