# 《未名舟》前030章强剧情链

## 0. 前30章目标

前三十章不讲大道理，只让读者记住六件事：

1. 地球被改名成舟材世界。
2. 妹妹卿禾被改成燃灯童。
3. 卿宸抢回她名字里的一笔。
4. 未名舟醒了。
5. 船上有饭、有药、有守线者、有老街。
6. 沈砚很强，但未名舟能反杀。

## 0.1 前30章惊喜钩总原则

前30章每一段都要有“得到但不白得”的钩子。读者先因为获得物、人或线索松一口气，下一刻发现它带着命名庭的钉子、账、香、索或选择题。

大纲写法固定追加：

意外钩：获得什么 / 表面用处 / 真正用处 / 代价 / 回收章。  
情绪钩：本章从哪种情绪推到哪种情绪，章末用什么钩住下一章。

前30章重点意外物链：燃灯童登记牌、伙料铜牌、同锅灶灰、半声到、三张未完伤情牌、二十九床腕带、铁牙旧守线编号、反挂钉、压舱者乳名、第一块求救木牌、假裂名定位钉、缝不回去的门、地球第一枚自命印。

## 1. 001-010：地球舟材化，未名舟醒

### 001 第一声改名

全球屏幕宣布：地球文明审定完成，新名“舟材世界”，三小时后拆解。命名庭驻地球官员当众演示改名：一个拒绝接受舟材名的普通人被改成“示材台”，一截手臂木化成讲台支架，屏幕标注用途为“展示舟材纹理”。卿宸先看见这种合法羞辱，才冲向卿禾。卿禾手腕出现“燃灯童”。卿宸在“燃灯童”里抢回“禾”的一竖，那一竖像断掉的麦秆，从灯芯纹里露出来。卿禾短暂醒来叫哥。章末，他抱着妹妹跌进黑水，摸到一截无名船骨。

### 002 无名船骨

黑水下全是城市残骸。船骨要求卿宸给它取名才救人。卿宸拒绝取名，把“禾”的一竖按进船骨。未名舟醒，只能载七十人。章末，燃灯架启动。

### 003 七十个名额

未名舟只有七十个名额。卿宸先救卿禾，错过三个孩子。七十名无票者登船，三个孩子被拆名官带走，成为童灯队伏笔。

### 004 拆名官沈砚

沈砚登场，要求归还燃灯童和无名船骨。他展示拆名刀和定名钉。卿宸挡住一次正式命名，但船骨开裂。沈砚救下另一批人，证明合法拆解也能减少死亡。

悲剧魅力埋针：沈砚袖口内侧露出一枚旧守灯人徽章。他曾见过无名者因为没有合法身份而彻底消失，所以相信“哪怕被命名成材料，也比无名消失强”。这不是洗白，只是让他的错更锋利。

### 005 归家灯

黑水夜航，船上无路。卿禾主动点亮残存燃灯名，归家灯亮起。未名舟找到短暂航路。代价是卿禾忘记一段和哥哥有关的糖水记忆。

### 006 船上缺粮

救上船的人开始挨饿。陆小满提出不按工具名分粮，先按伤者、孩子、劳力登记。章末，船底发现一袋写着“伙料专用”的米。

意外钩：米袋不是单纯粮食，袋口铜牌背后有烧黑的“周记夜饭”旧刻，表面是开粮钥匙，真正是老周旧名锚点；代价是米、锅、人被铜链拴成一套伙料资产，回收至007、008。

情绪钩：饥饿恐慌 -> 看见配给的惊喜 -> 发现按用途分饭的愤怒 -> 找到米和活人的下一章期待。

### 007 伙料老周

米袋旁藏着厨子老周，手腕写着“伙料”。卿宸从他铁锅旧刻痕里抢回“周”的一撇。章末，老周说要吃饭，得先烧穿锅底的命名铜牌。

意外钩：意外获得老周和铁锅，表面能开米袋，真正开启灶火舱和乡愁面锚点；代价是锅底铜牌不穿，饭会变成按用途发放的配给，回收至008、009。

情绪钩：警惕陌生人 -> 厨子嘴碎带来的松动 -> 抢回一撇的爽 -> 锅底铜牌的新难题。

### 008 第一顿非法粥

老周煮出第一锅粥。卿宸把“燃灯童”登记牌塞进灶火压锅底。七十人第一次有活下去的实感。老周下意识说这锅还不算饭，真正的饭要用真名喊人来吃，埋下“乡愁面”私密锚点。粥香引来沈砚追踪舟。

意外钩：燃灯童登记牌被当柴压锅底，表面烧穿锅底铜牌，真正把燃灯资产转成灶火舱第一口人饭；代价是追踪香启，粥香反成敌方定位线，回收至009、010。

情绪钩：能不能生火的急 -> 第一口热粥的暖 -> 追踪香亮起的不安。

### 009 归还燃灯童

沈砚要求归还燃灯童和伙料，称这是合法物资。卿宸拔松定名钉，老周没有被拖回补给箱。代价是未名舟被判为拒名船。

### 010 非法开航

全船选择：交人换合法停靠，或跟卿宸走黑航道。卿宸不能强迫别人留下，只摆出空白船头木牌。大多数人留下，少数人投靠沈砚，并带走航迹。章末，童灯队名册出现。

意外钩：童灯队孩子抢回半声“到”，表面只是打断嗅灯，真正给船头空白木牌留下第一枚回应痕；代价是未名舟被判拒名船，童灯队债务坐实，回收至011、018-020。

情绪钩：被追十息的急 -> 半声到的疼爽 -> 全船选择的热 -> 黑航道里撞见药箱的新奖励钩。

十章结算奖励：未名舟醒、七十名无票者登船、归家灯初亮、灶火舱初亮、拒名船身份成立。读者必须明确：主角不是在逃，他已经抢下第一条不归命名庭管的船。

## 2. 011-020：抢医舱、兵骨、压舱者

### 011 药箱醒来

一只被拆成舟材的急救方舱漂来，里面只剩药箱、袖章和三张伤情牌：一张空白，两张只露出未写完的旧牌角。苏晚醒来，手腕写着“药箱”。卿宸用她袖章上的旧名引裂，苏晚稳住重伤者。

意外钩：三张伤情牌，表面是记录伤者，真正能拒绝“已尽力”结案并追踪童灯队、二十九床等未救线索；代价是每写一张，回声医舱就会听见更多未救者索债，回收至012、013。

情绪钩：黑航道撞箱的警惕 -> 孩子喘不上气的急 -> 苏晚醒来救人的稳 -> 急救方舱远去的追击钩。

### 012 抢急救方舱

未名舟第一次主动抢资源，目标不是整座医院，而是即将被拖走的急救方舱。卿宸、老周、陆小满配合，抢回回声医舱雏形。

意外钩：抢回的不是完整方舱，而是十三个活伤者和二十九床腕带，表面是救援成果，真正是欠命账定位器；代价是五十四个未救床号挂上船，回收至013和后续追债线。

情绪钩：主动抢救的爽 -> 十三红点亮起的紧 -> 抢回十三的胜 -> 未救五十四的疼。

### 013 欠命账

医舱救人，但死者索债声出现。苏晚立规：救命债不能删，只能还。

意外钩：结案章碎片，表面能换取现存伤者稳定，真正是命名庭删账机制的证据；代价是未名舟拒绝结案后欠命账拖慢航行，回收至015反挂钉。

情绪钩：被索债的冷 -> 拒绝已尽力的硬 -> 唐守义未删的痛 -> 追名兵登船核账的危险。

### 014 兵料铁牙

断臂老兵铁牙被登记为兵料。卿宸抢回“铁”的一横，兵骨舱初生。铁牙断臂骨茬第一次与船骨共振，他发现这条船不像武器，更像他缺掉的另一半身体。

意外钩：铁牙旧守线编号藏在黑铁牙套内侧，表面是旧役残号，真正能提前听见登船索试力和核账坞方向；代价是编号一响，追名兵也能确认兵骨舱雏形，回收至015、020。

情绪钩：追名兵核账压迫 -> 断臂老兵醒来的意外 -> 抢回铁一横的爽 -> 登船索预挂的危机。

### 015 登船索

追名兵登船，把船民改成临时船工。铁牙守线，老周用灶火烟恢复人味，登船索成形。

意外钩：登船索预挂钉被反挂，表面阻止拖船，真正让命名庭下次拖船前必须背过未名舟全部杂账；代价是登船战提前一刻，回收至016-020。

情绪钩：全船被拖走的急 -> 杂账一件件压上索的热 -> 反挂成功的爽 -> 登船战提前的紧。

### 016 压舱者

未名舟遇到被压进正规方舟底部的人。沈砚拿出保命书，证明有人曾自愿签名换家人上船。卿宸不能强救所有人。

### 017 保命书

压舱者本人和家属意见相反。卿宸停止抢笔，听他们说完。救下伪造签名者，放走真正自愿压舱者。

### 018 陆小满的问题

卿禾想主动点灯探路，卿宸硬拦，怕她再失去记忆。结果未名舟错过短路，差点撞进沈砚布下的假安全航标。陆小满追问：你是怕她受伤，还是怕她不再需要你保护。卿宸承认自己先救妹妹，错过三个孩子，也第一次意识到他把妹妹当成“只能由自己救的人”。章末，卿禾挣开他的手，自己点亮半盏灯，童灯队在灯外出现。

### 019 童灯队

三个孩子已被训练成燃灯武器。卿宸没能救下他们，只抢回半声哭。卿禾点灯过度，忘记旧校牌。

### 020 黑航道

全船第一次协作：老周稳饭，苏晚稳伤，铁牙守线，卿禾点灯，卿宸掌舵。未名舟冲进黑航道，章末江城旧钟响起。

二十章结算奖励：回声医舱、兵骨舱、登船索、压舱者乳名、童灯队债务同时成立。读者必须明确：这不是单人英雄，而是一条船开始长出队伍打法。

## 3. 021-030：老街、市井舱、第一条船规

### 021 江城钟声

黑航道里浮着江城老街碎片。云织登场，手腕写着“缝材”。她用最后一根缝线接住老街和未名舟。

意外钩：云织缝住的第一件东西不是整条老街，而是一扇缝不回去的门；表面能挡黑水，真正是她私密锚点和市井舱门槛，代价是门后藏着旧街失踪名单，回收至026、029。

### 022 船内公议

卿宸私心、童灯队债、压舱者争议一起爆发。公议不是审判主角，而是全船第一次决定：未名舟不能被任何人私自命名。

### 023 第一条船规雏形

卿宸提出：不强救，不替名，但不放弃追问。船民暂时接受。沈砚送来合法港口坐标，只要船有名字即可入港。

补强要求：本章结尾要出现“第一块临时求救木牌”。可以是老街碎片上幸存者用菜刀刻出“救我娘，不要救我铺子”，挂到船头空白木牌下。它不是正式承约制，只是承约墙的种子，让平台感提前埋入。

正义陷阱埋针：求救木牌旁同时救上一名陌生人，他手腕上有一道看似真实的裂名。未名舟按规则先救，不逼问真名。读者先感到这条规则很爽，后面再发现沈砚把定位钉藏在假裂名里。

意外钩：第一块求救木牌，表面是普通人求救，真正是承约墙种子；代价是假裂名者随木牌上船，定位钉开始锁定市井舱，回收至027、028。

### 024 合法港口

缺粮缺药，合法港口就在眼前。命名派出现。卿禾问，如果取名能让大家活，为什么不取。

### 025 拖城索

卿宸决定先拖回江城一段，而不是立刻入港。压舱者乳名、钟楼铜钟、云织缝线共同引裂城名。拖城索成形。

### 026 老街上船

菜市场、旧钟楼、半条巷子成了市井舱雏形。船上第一次像家。代价是老街底下藏着沈砚定位钉。

意外钩：旧钟楼铜钟不是单纯地标，钟腹里有地球第一枚自命印的回声；表面稳住老街，真正指向卷级资产；代价是钟声也会放大定位钉，回收至027、030。

### 027 沈砚的离间

沈砚公开卿宸错过童灯队的记录，逼船民互相定名、互相撤权。命名派要求给船取名，交出燃灯童，换合法港口。与此同时，023章救上来的陌生人手腕假裂名发热，定位钉锁住市井舱边缘。沈砚证明：未名舟不逼问，也可能被人利用。

### 028 公议反杀

沈砚想把公议变成清算卿宸的刀，陆小满反手把问题改成“谁有权替全船取名”。卿宸不把陌生人拖出来审，而是当场抢回一名命名派船民被暗钉的半笔真名，并让假裂名定位钉显形。公议升级出“自愿陈述权”：被救者可以说出自己知道的危险，未名舟不得逼供，也不能把不问当成没风险。公议变成反击定名术。

### 029 市井舱保卫战

童灯队和追名兵攻入市井舱。卿宸、铁牙、苏晚、云织、老周首次完整配合，守住市井舱，抢回一个童灯队孩子的乳名一笔。

### 030 未名舟第一规

未名舟确立第一条正式船规：不替人命名，不强迫登舟，不删除债。卿宸重新掌舵，但不再单独决定。章末，江城旧钟指向地球第一枚自命印。

意外钩：地球第一枚自命印只出现半枚印泥，不出现完整神器；表面是卷级线索，真正证明普通世界也曾自命；代价是命名庭同步锁定印泥气息，第一卷主竞速正式开启。

三十章结算奖励：市井舱、拖城索、第一块求救木牌、第一条船规、地球第一枚自命印线索成立。读者必须明确：未名舟不是避难所，它会接活、救人、打仗，并把规则越打越硬。

## 4. 前30章结算

已获得资产：未名舟、归家灯、灶火舱、回声医舱、兵骨舱、市井舱、登船索、拖城索、第一条船规。

已欠债务：童灯队、卿禾记忆损耗、压舱者选择、合法港口诱惑、沈砚航迹、地球第一枚自命印。

前30章读者必须得到的感觉：这本书设定新，但一点不虚；每一章都有人、饭、灯、船、敌人和下一口气。

## 5. 前30章主动升级口径

前30章禁止写成“主角被迫解释自己为什么正确”。每一次争议都要迅速变成敌我动作。

陆小满的追问，不是刹车，是把敌方陷阱翻出来。  
卿宸的承认，不是认输，是让下一次抢笔更准。  
船规的建立，不是会议，是战前装甲。  
求救木牌的出现，不是制度讲解，是普通人把命挂到船头。
