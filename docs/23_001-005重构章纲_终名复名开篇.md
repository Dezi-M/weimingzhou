# 001-005 重构章纲：活拆地球与第一盏归家灯

本文件用于当前 001 起大修。它服从 `22_001-200全书地基终审_末世生存战斗爽文版.md`，但不当硬门禁。若正文写出来更有力，以正文为准。

## 输入复核

- 当前阶段：001 起重构，旧稿 001-005 只作素材池。
- 本段核心承诺：地球被拆成货，卿宸开船去抢回家。
- 底层战争：终名 vs 复名。敌人把人、物、城市锁成最终用途；未名舟让它们还能改变。
- 001-005 明面任务：活拆地球入场，卿禾燃灯，卿宸第一次抢笔，无名船骨醒，七十人登船，沈砚有效慈悲登场，卿禾主动点归家灯。
- 本段结算：未名舟还不是舰队，只是一条能载七十人的破船；但它已经抢回第一组地球组件：一笔“禾”、无名船骨、七十个活人、归家灯方向。

## 001 第一声改名

### 八格驾驶舱

1. 危险：江城地铁卡在江底隧道，广播宣布地球原名撤销，新名“舟材世界”；同一时间，一间急救室连病床、输液架和医生袖章被吊上空中押运，地铁轨被抽成航道轨。
2. 目标：卿宸必须从地铁里逃出，赶到江城一中北门，在卿禾被点成燃灯童前抢回她名字里的一笔。
3. 敌压：白制服拆名官公开把外卖员赵成改成“示材台”，校门十二个孩子被圈入童灯格，卿禾手腕出现“燃灯童”。
4. 行动：卿宸砸地铁门、钻黑水、接出一个“童料预备”的孩子、沿抽轨后的地铁骨跑向学校、冲过金线，伸手从“燃灯童”里硬抢“禾”的一竖。
5. 战利品：半截“禾”的竖画；无名船骨第一次被这笔旧名敲醒。
6. 损失/代价：卿宸右手被“灯灰”临时名烫伤；校门另外十一只童灯位仍亮；糖水摔碎，卿禾旧日常开始缺口。
7. 感官：急救室输液架撞风的叮当声；黑水里的铁锈和旧纸味；糖水椰果滚在碎玻璃上的软响；“燃灯童”烧皮肤的焦味。
8. 关系：卿禾第一次从灯芯痛里醒来，第一句话不是感谢，而是问糖水；卿宸的护妹欲成立，但也埋下他容易替她挡选择的隐患。
9. 宏大线：地球不是背景，而是被拆开的主角；无名船骨不是逃生神器，而是日后劫回地球组件的第一块骨。

### 意外钩

- 获得什么：半截“禾”的竖画。
- 表面用处：让卿禾从燃灯童点火里短暂醒来。
- 真正用处：旧名笔画能敲醒无名船骨，证明命名庭的终名并非不可撬动。
- 代价：卿宸被临时写成“灯灰”，右手留下烧伤。
- 旧账折痕：糖水记忆缺口和十一只空灯位，后续牵出童灯队债务。

### 情绪钩

正常傍晚 -> 活拆地球的惊与恨 -> 抢回“禾”的燃 -> 黑水里船骨醒的紧。

### 开写提醒

第一屏别讲世界观，直接写“急救室被吊走”或“地铁轨被抽走”。赵成示材台保留，但不要让它变成新闻插播太久；它要像一刀，切完立刻奔向卿禾。

## 002 无名船骨

### 八格驾驶舱

1. 危险：整条校门街塌进黑水，活人、校门、广告牌、地铁车门和医院残件一起往拆料单里沉。
2. 目标：卿宸要让船骨醒来，但不能用命名庭的方式给它定死用途。
3. 敌压：船骨空白处反复浮出“请命名”；命名后可载两人、七人、七十人，像救命诱惑，也像另一枚钉子。
4. 行动：卿宸拒绝命名，先把课桌踢给溺水男生，再用卿禾半截“禾”贴住船骨；校门、广告牌、地铁车门、梧桐树和断牌互相咬合成船。
5. 战利品：未名舟雏形；第一批七十个白痕名额。
6. 损失/代价：第七十一个眼镜老师没能上船，只把点名册丢上来；船骨留下“只能七十”的裂口。
7. 感官：黑水压肺的闷痛；老木头晒过的旧暖；广告牌翻成甲板的刮响；白痕一条条刻进船舷的声音。
8. 关系：大叔从骂卿宸疯，到第一个拉人上船；普通人不是旁观者，开始参与救援。
9. 宏大线：未名舟的第一条存在方式成立：不靠定名醒来，而靠旧物和活人互相拉住。

### 意外钩

- 获得什么：无名船骨醒成破船。
- 表面用处：救七十人离开黑水。
- 真正用处：证明“无终名”也能组织物件成船，是复名文明的第一粒种子。
- 代价：只能七十；眼镜老师沉没，留下泡烂点名册。
- 旧账折痕：点名册上的孩子旧名，后续成为童灯队反抢的锚。

### 情绪钩

窒息 -> 诱惑命名 -> 拒绝后船醒的惊 -> 七十白痕的痛。

### 开写提醒

这里不要把“拒绝命名”讲成哲学。让读者先看见：卿宸拒绝写名字，反而踢课桌救人，船由旧物自己拼起来。

## 003 七十个名额

### 八格驾驶舱

1. 危险：燃灯架在船头自动亮起，卿禾被第一灯位拖拽；黑水外还有三个孩子趴在校门碎片上，船却只能载七十。
2. 目标：卿宸必须守住“船上人不是物资交换”的底线，同时尽力抢回孩子的旧名。
3. 敌压：女官用湿点名册点出许愿、陈一桥、梁朵，准备补入童灯队；船上有人提议把卿禾交出去换三个孩子。
4. 行动：卿宸按住燃灯架，跳出船舷去抢孩子；大叔甩电缆救卿宸；老师留下的点名册纸角短暂停住“灯”字；最终三个孩子被拖走，但其中“许愿”的印记缺了一角。
5. 战利品：许愿名字里一点旧愿痕；童灯队债务正式落下。
6. 损失/代价：三个孩子被临编童灯队带走；卿宸第一次明确救慢了；卿禾听见自己被当成可交换物。
7. 感官：粉色水杯掉进黑水的闷声；湿点名册翻页声；电缆勒进卿宸腰侧的痛；燃灯架空灯位的焦木味。
8. 关系：卿宸护卿禾很凶，但卿禾开始意识到自己不能永远被藏在哥哥身后；大叔正式从路人变成船上第一批硬嘴帮手。
9. 宏大线：未名舟第一笔债不是规则债，而是没救回的三个孩子；后续童灯巡舟的种子从这里埋下。

### 意外钩

- 获得什么：许愿旧名的一点“愿”痕。
- 表面用处：临时停住一个灯字。
- 真正用处：后续可作为童灯队反烧的第一枚旧名锚。
- 代价：只抢回一点，没抢回人。
- 旧账折痕：粉色水杯、湿点名册、缺角印记。

### 情绪钩

获救余悸 -> 船上交换争吵 -> 卿宸跳水抢人燃起希望 -> 三个孩子被带走，恨意压住。

### 开写提醒

这一章不要让卿宸全胜。读者要疼一下，知道七十名额不是设定，而是欠账。

## 004 沈砚登场

### 八格驾驶舱

1. 危险：沈砚黑舟抵达，白舟救下远处一栋楼顶的人，给药、毯子、临时舱籍都是真的。
2. 目标：卿宸要看清沈砚的有效慈悲，并阻止他把未名舟定成“临时逃材舟”。
3. 敌压：沈砚递出真实诱惑：归还卿禾、船骨和旧名，七十人可转临时乘员，有饭、有床、有药；同时以“应急浮材债”救人，七日不还就转舟材补偿。
4. 行动：沈砚用救援钉搭浮道救人；船上动摇；沈砚把定名钉钉进梧桐桅杆；卿宸用许愿旧名纸角烫断红线，强拔定名钉。
5. 战利品：断掉的定名钉；沈砚第一次确认卿宸能看见裂名。
6. 损失/代价：强拔定名钉裂船骨，黑水灌甲板；船民对合法救援的动摇更深。
7. 感官：黑伞无声接雨；定名钉合页般入水；白舟干净毯子上的药味；红线割手的热痛。
8. 关系：卿宸不再只对抗恶人，而是对抗一个真的救人的秩序；西装男人成为船上诱惑线的活口。
9. 宏大线：终名文明首次露出复杂面，命名庭不是只会杀，它会救人，也会把救命写成债。

### 意外钩

- 获得什么：半枚断定名钉。
- 表面用处：证明沈砚定名失败。
- 真正用处：后续可反查命名庭的舱籍、债籍与救援钉体系。
- 代价：船骨裂，船民开始知道“合法港真的能救命”。
- 旧账折痕：应急浮材债七日线，后续可以成为诱惑和倒计时。

### 情绪钩

恨敌人 -> 发现敌人真会救人 -> 船上动摇 -> 卿宸拔钉险胜但船裂。

### 开写提醒

沈砚不能像坏人上来放狠话。他要先救人，而且救得漂亮。读者要短暂理解西装男人为什么想过去。

## 005 归家灯

### 八格驾驶舱

1. 危险：沈砚退走后，未名舟无舵、无帆、无航图，黑水只露出合法浮道、债籍转港、燃灯回收线，没有自由活路。
2. 目标：卿禾主动点亮归家灯，带船脱离合法航标，但不能被再次写成燃灯童。
3. 敌压：白舟远处仍有饭、床、药的暖光；船内有人饥冷动摇，认为不如回合法港。
4. 行动：卿禾按住第一灯位，不让卿宸替她决定；灯芯里浮出一粒椰果旧光，照出糖水店、学校北门、旧公交站等归家残影；全船压裂缝、塞湿课本、堵船舷，沿歪路冲出黑水夹缝。
5. 战利品：归家灯初亮；第一条不靠命名庭航标的短路。
6. 损失/代价：卿禾失去关于“给谁买糖水”的一小段记忆；船仍缺粮，第一口饭钩子落下。
7. 感官：椰果在灯芯里晃的黄光；黑水夹船的闷雷；湿课本塞进木缝的纸浆味；孩子肚子叫的尴尬声。
8. 关系：卿禾第一次明确说“别把我当只能被你保护的人”；卿宸第一次被妹妹反向拉住。
9. 宏大线：归家不是回到原点，而是照见被拆走的地球旧方向；后续抢回家，不再只是逃命。

### 意外钩

- 获得什么：归家灯。
- 表面用处：照出一段能走的路。
- 真正用处：追踪被拆走的地球组件旧方向，是后续劫回医院、学校、钟楼、市井的导航种子。
- 代价：卿禾糖水记忆缺一块。
- 旧账折痕：椰果灯芯、少冰/多椰果谁喝，后续作为卿禾记忆回收线。

### 情绪钩

冷饿动摇 -> 卿禾主动点灯 -> 船冲出窄路的松 -> 肚子叫和船底空锅声，转入缺粮。

### 开写提醒

这一章是卿禾主动性立住的关键。不要把她写成“牺牲灯芯”，而是写成她自己决定用灯找路，同时付出记忆代价。

## 手感掂量

如果 `001-005` 写得太满，读者会累。每章只守一个发动机：001 抢禾，002 船醒，003 七十之痛，004 沈砚有效慈悲，005 卿禾归家灯。其他设定只露边角。

## 软规则预检

- 可顺手看：001 的活拆地球要选一个主画面写透，急救室、地铁、学校不要全都平均铺。
- 我会留意：沈砚要先救人再伤人，这样诱惑才立得住。
- 也许可以：005 的小幽默只放一个，比如湿课本塞船缝或孩子肚子叫，别冲淡卿禾失忆。
