# 《未名舟》角色能力、军团与承约制

## 1. 主角组

### 卿宸

定位：抢笔者、掌舵人、正义规则扩张者。

能力：见裂、抢笔、写骨。

爽点：他不是求原谅的人，而是把敌人的命名规则当场打穿的人。

限制：不能凭空造名，不能替人决定，不能用“我是为你好”强行登舟。

称号成长：

1. 拒名船主：第一卷被敌人通缉时出现。
2. 抢笔者：抢回多个真名笔画后被船民传开。
3. 未名承约人：承约墙成形后出现。
4. 万界拒名第一人：打穿命名庭后出现。

称号必须从剧情里长出来，先由敌人骂，再被船民接住，最后变成万界传闻。

### 卿禾

定位：妹妹、归家灯源、燃灯童反杀者。

早期能力：点亮归家灯，找回短路，短暂唤醒被燃烧者的旧声。

成长方向：从被救者成长为能主动点灯、反烧名册、召回失路者的核心角色。

硬规则：卿禾不能只是代价容器。每20-30章至少有一次主动选择或主动反杀。

### 陆小满

定位：普通人第三立场、追问席。

作用：她不是拖主角后腿，而是把“未名舟正义”校准成更能长期打仗的规则。

硬规则：陆小满的追问必须带来行动升级，不能连续变成内耗。

主动欲望：她不想成为主角的“良心提醒器”，她要证明普通人也能制定让强者必须遵守的规则。后期她应拥有自己的追问席队伍。

### 苏晚

定位：回声医舱主责。

能力：救命、记债、拒绝把病人变成材料。

爽点：她能让“救回来的人不是欠主角命，而是重新拥有还债和活下去的资格”。

主动欲望：她要建立一套不把病人卖给命名庭、也不把病人绑在未名舟上的救命规则。

出场物件：急救方舱、袖章、药箱、伤情牌。避免回到旧版“医院死亡通知”气质，医疗线必须服务救援、战斗和回声医舱成长。

### 老周

定位：灶火舱主责。

能力：让船上有饭，让被工具化的人重新闻到人味。

爽点：一锅粥比一百句设定更能抓读者。

主动欲望：他想让船上没人再被叫成食材。后期灶火队不只是做饭，还要负责粮线、士气和烟火反制。

### 铁牙

定位：兵骨舱主责、守线者。

能力：训练守线小队，抵挡追名兵。

风险：兵骨舱也可能把人重新压成兵料。

主动欲望：他要训练一支不会把弱者推到身后的守线军。

### 云织

定位：缝材反杀者、市井舱建造者。

能力：缝合残城、残船、残路，但缝错会长出不属于本地的墙。

爽点：她能把被拆掉的城市缝回船上，让家不是概念，而是菜市场、钟楼、巷口。

主动欲望：她不想只修补破烂，她想证明残城也能重新长路。

### 迟灯

定位：买名市内线、偷价者、假背叛专业户。

能力：偷换价格、伪造低价、把敌人的买名契反挂到敌人身上。

爽点：他让市场线有谍战和反转，不只是砸拍卖场。

风险：他太会说谎，读者和伙伴都不能完全信他。

## 2. 第一阶段敌人

### 沈砚

定位：拆名官，第一卷主敌。

强点：他不是只会杀。他能证明正规拆解救的人更多，能拿合法港口、医疗、粮食诱惑船民。

打法：不急着摧毁未名舟，而是让船民自己给船取名、交人、撤权。

被反杀方式：未名舟每次证明“合法命名不是唯一活路”，他就失去一块话语权。

## 3. 承约制

承约制不是游戏系统，而是未名舟长大后的平台规则。

一个人、一队人、一座城或一个世界可以把求救牌挂上承约墙。未名舟不替他们决定，只承接明确任务。

任务类型：

- 救名约：抢回某人的一笔真名。
- 拔钉约：拔掉敌方定名钉。
- 寻锚约：找回旧物、旧路、旧称。
- 护航约：护送无名者穿过合法航道。
- 猎价约：反杀买名商路。
- 打捞约：从黑水里捞回残城、残船、残神。
- 医债约：救命并记录真实代价。
- 反追约：引诱追名军进入未名舟设好的战场。

承约制成长阶段：

1. 求救木牌：前30章出现，只是普通人把命挂到船头。
2. 临时承约：第一卷中段，船上开始按具体目标接活。
3. 承约墙：第二卷成形，可接外世界任务。
4. 承约榜：第三到第四卷成形，队伍、报酬、风险公开。
5. 万界承约网：第七卷后成形，与合法承约厅正面对抗。

承约制不能写成系统面板。每一次承约都要有物件：木牌、布条、断钉、旧票、半块饼、血字、城印、庙匾。

## 4. 军团结构

早期是精英小队，中期是职能队，后期是军团。

基础职能：

- 灶火队：后勤、粮线、烟火反制。
- 回声医队：救援、伤病、欠命账。
- 守线队：登船战、防线、压制追名兵。
- 缝城队：修船、拖城、临时阵地。
- 归家灯队：找路、召回、反烧名册。
- 夜航队：侦察、偷渡、黑航道突进。
- 承约队：接任务、分账、处理退约和违约。

组合技示例：

- 灶火烟 + 归家灯：让被定名者短暂想起自己吃过的一顿饭。
- 登船索 + 兵骨阵：把追名兵从合法甲板拖进未名舟战场。
- 缝城线 + 市井舱：把残街变成临时堡垒。
- 回声医舱 + 追问席：救人之后不洗掉代价，让敌人无法用“债务清零”收买。

## 5. 平台正义爽

未名舟越像平台，越要证明自己不是新锁链。

敌人会模仿承约制，推出更安全、更高报酬、更合法的承约厅。未名舟的反击不是自我怀疑，而是把换岗、拒约、退约、分债、不卖名打成更强的平台规则。

核心口号：正义不是喊出来的，是能接活、能分钱、能退出、能救人、能打胜仗的。

## 6. 敌方经济对抗表

| 敌方商品 | 敌方诱惑 | 未名舟反击 |
|---|---|---|
| 合法港口 | 有饭有药有保护 | 抢废港，建临时港，不卖船名 |
| 岗位名 | 给工作和身份 | 换岗牌，让人可以退出旧岗位 |
| 亲缘凭证 | 证明谁是谁的亲人 | 旧物锚点和本人回应优先 |
| 免拆资格 | 暂时活下去 | 承约救名，公开代价，不拿人换资格 |
| 高薪承约 | 挖走精英队 | 公开承约、退约权、分账锅 |
| 王庭赐名 | 合法化和安全 | 拒赐名印，不称王也能调军 |

每个大阶段至少让敌方经济赢一次小局，否则敌人不够有压迫感。

## 7. 承约墙可视化规则

承约墙必须让读者看得见船在长大。

第二卷开始，每接一个承约，墙上多一块发光木牌，烧焦字写明任务、报酬、风险和退约条件。

任务完成，木牌变成暗金色，挂回墙上，成为已履约荣誉墙。

违约或失败，木牌裂开，但不会消失，提醒所有人代价。

每个新伙伴上船，第一眼都要看承约墙，并用自己的反应说明他信不信这条船。

## 8. 组合技口号

组合战必须让听书读者秒懂。

示例：

- 烟是路，灯是眼，烟灯一起走，死人也能回头。
- 缝城线加市井舱，破街变城墙。
- 登船索落地，兵骨不退线。
- 药箱记债，活人还账。

每场组合战前后，至少有一个角色用一句土话、狠话或口号概括打法。不能只写术语。

## 9. 普通人高光清单

每50章至少一次普通人高光，且必须改变战局。

可用方向：

1. 厨娘烧掉自己仅剩乳名，不让沈砚用她的名册定位灶火舱。
2. 被救老人拒绝合法港口床位，留下来守第一块求救木牌。
3. 孩子把半块饼压在承约牌下，说报酬只有这个，但求你救我娘。
4. 曾经投靠沈砚的人回船，不求原谅，只带回一枚定名钉的弱点。
5. 普通船民在军团战里不参战，只把饭准时送到守线队手上，因此防线没断。

普通人高光的原则：不靠超能力，靠选择、旧物、饭、路、亲人和胆气。

## 10. 角色私密锚点

主要角色不能只承担功能，还要有一个极私人的小执念。它可以不宏大，但要能反复回收，让读者觉得他们是活人。

### 卿宸

私密锚点：他记得卿禾小时候总把自己的名字写歪，他一直想教她把“禾”写正。后来他发现，卿禾不需要他教正，她要自己写。

### 卿禾

私密锚点：她想收集船上所有人写错过的名字，做一本错字本。错字本不是笑话，而是证明每个人都有没被命名庭规范过的一面。

### 老周

私密锚点：乡愁面。他想做一碗只有用真名喊人才入味的面。后期这碗面可以成为婚礼、归船、战后庆功和送别的固定符号。

### 苏晚

私密锚点：药柜标签。她给每个抽屉贴的不是药名，而是“先止血”“先喘气”“先睡一会儿”这种人话，拒绝把病人写成药材用途。

### 铁牙

私密锚点：断臂龙骨共振。他守船不是因为纪律，而是因为船骨声像他失去的那截手臂在回应他。

### 云织

私密锚点：一扇缝不回去的家门。她每次缝城，都在找那扇门到底通向谁家。

### 陆小满

私密锚点：她随身带一小截铅笔，专门记录“强者说得太满”的句子。不是为了嘲笑，是为了日后让规则别把人压扁。

### 沈砚

私密锚点：旧守灯人徽章。他曾经守过一盏给无名者登记的灯，但那盏灯没等到合法文书就灭了。从此他相信，错误的名字也比没有名字强。

## 11. 第一卷反派梯队

第一卷不能只有沈砚一个人压场，否则所有压力都会挤到一个反派身上。

| 反派层级 | 功能 | 出场作用 |
|---|---|---|
| 驻地球命名官 | 开局恨点 | 当众把拒名者改成示材台，证明命名庭的恶是合法羞辱 |
| 追名兵队长 | 动作压迫 | 登船、钉甲板、制造近身战 |
| 正规方舟管事 | 诱惑压迫 | 提供饭、药、床位、合法身份，逼船民动摇 |
| 沈砚 | 哲学主敌 | 证明合法拆解短期有效，并主动设正义陷阱 |
| 童灯队 | 情绪敌人 | 让卿宸的旧错变成会攻击船的疼点 |
| 命名庭远影 | 卷末压力 | 不正面下场，但用榜单、判决和收益表压住地球 |

每30-50章至少有一个具体反派脸压场，不能只写“命名庭”这个抽象组织。
