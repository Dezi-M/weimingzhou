# 《未名舟》文档使用索引

本索引只服务当前写作。顶层文档保持精简；已完成的一次性校验、已失效执行章纲和被合并的脑暴文件已清除，不再作为日常入口。

当前最高口径：从 001 章起允许大修。重构后不翻已失效路线，不让失效文本继续当方向盘。

默认创作方向：

> 地球被拆成货，卿宸开船去抢回家。
> 《未名舟》不是规则题，也不是普通末世逃亡；它是命名权末世里的抢人、抢水、抢药、抢旧物、抢街区、抢地球组件的生存战、守船战、夺名战、聚船战和立岸战。
> 更底层地说，它写的是“终名 vs 复名”：一个把万物固定成最佳用途的宇宙，遇见了一群宁愿低效、疼痛、漂泊，也要保留改变余地的人。
> 终篇级钩子是：命名庭偷的不是地球旧物本身，而是地球和万界那些还没结束、还能改口的明天。

## 冲突取舍

当文档之间冲突时，按这个顺序取舍：

1. 正文已经重构后成立的事实。
2. `22_001-200全书地基终审_末世生存战斗爽文版.md`。
3. `14_当前故事状态账本.md`。
4. `伏笔索引.md`。
5. 当前阶段补纲；已完成阶段补纲只作回查，不压过当前账本。
6. `30_主创终审合订_大纲外扩十章主菜与偷明天.md`。
7. `创作复盘_常见问题与改进.md` 的最新条目。
8. 角色、军团、宏大架构、脑洞文档。
9. 已清除路线不再参与取舍；若现行路线需要某类桥段，按新基座重新设计。

一句话：重构就是换航道，已失效路线不再上桌。
