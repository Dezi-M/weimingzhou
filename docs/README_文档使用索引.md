# 《未名舟》文档使用索引

本索引只服务当前写作，不保留已被推翻的阶段补丁。

当前最高口径：从 001 章起允许大修。不要再按旧阶段补丁前提续写。

默认创作方向：

> 地球被拆成货，卿宸开船去抢回家。
> 《未名舟》不是规则题，也不是普通末世逃亡；它是命名权末世里的抢人、抢水、抢药、抢旧物、抢街区、抢地球组件的生存战、守船战、夺名战、聚船战和立岸战。
> 更底层地说，它写的是“终名 vs 复名”：一个把万物固定成最佳用途的宇宙，遇见了一群宁愿低效、疼痛、漂泊，也要保留改变余地的人。

## 冲突取舍

当文档之间冲突时，按这个顺序取舍：

1. 正文已经重构后成立的事实。
2. `22_001-200全书地基终审_末世生存战斗爽文版.md`。
3. `14_当前故事状态账本.md`。
4. 当前执行章纲，例如 `23_001-005重构章纲_终名复名开篇.md`。
5. `创作复盘_常见问题与改进.md` 的最新条目。
6. 角色、军团、宏大架构、脑洞文档。
7. 旧正文和旧章纲里的可用素材。

一句话：旧稿只能当素材，不能当方向盘。
