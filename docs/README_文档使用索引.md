# 《未名舟》文档使用索引

本索引是当前文档目录入口。AI / Codex / 自动审稿开工时，先读 `AI扫描入口.md`，再回到本索引判断加载范围。新大纲是唯一方向盘，文档只保留能继续写正文的活内容。

## 当前最高口径

一句话：

> 地球被拆成货，卿宸开船去抢回家。

更深一层：

> 命名庭偷的不是旧物本身，而是地球和万界那些还没结束、还能改口的明天。全书底层战争是终名 vs 复名。

039 后新主创口径：

> 高位面把万物压成唯一用途。地球把万物打回万种活法。
> 人先不跪，物才应阵。人间，开阵。

若接已发 038 继续，从无形脚印自然长出猎足踩影和首次人间开阵；若做全书新版，则从 001 起重埋“终相猎场 vs 人间万象”。039 后新增战场素材只读 `06_039后新素材库_人间开阵.md`，但已发 001-038 的人物、旧物、伤口、债和章末钩仍是连续性地基。

普通续写默认只读：

## 冲突取舍

当文档之间冲突时：

1. 正文已经成立的事实优先。
2. `AI扫描入口.md` 和 `14_当前故事状态账本.md` 优先于其他说明。
3. `00/01/03/04/05` 五份 active 主文档负责总纲、规则、路线、体系和审计；039 后素材另以 `06` 为准。
4. 039 后新增素材以 `06_039后新素材库_人间开阵.md` 优先，001-039 已成立正文事实优先级仍高于素材库。
5. `伏笔索引.md` 只管活伏笔，不替代阶段纲。

一句话：日常创作只看活文档，缺新敌招、新战法、新素材就在新大纲里现造；已发正文事实不现造、不抹掉。
