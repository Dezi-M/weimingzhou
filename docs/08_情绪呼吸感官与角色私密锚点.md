# 《未名舟》情绪呼吸、感官系统与角色私密锚点

## 1. 本轮自省

前一轮基建已经解决了长线骨架、敌方经济和收集追读，但仍有一层不足：温度。

如果只有正义一直爽，没有呼吸感，读者会兴奋但不一定留下。真正让人跟两千章的，是他想知道船上的人今晚吃什么、谁还没睡、谁把旧物藏在枕头下。

## 2. 情绪缓冲带

每50章安排1-2章低压日常或半日常章节。

低压章不拖节奏，它承担三件事：

1. 让船像家。
2. 让角色像人。
3. 让下一场大战有更疼的东西可失去。

可写内容：

- 老周研发新菜。
- 卿禾教孩子们认字。
- 苏晚整理药柜。
- 铁牙夜里听断臂和船骨共振。
- 云织缝一扇永远缝不回去的门。
- 陆小满记下强者说得太满的话。
- 卿宸在船头听旧名低语。

## 3. 元视界感官

命名不是抽象规则，而是一套读者能看见、听见、闻到的异能体验。

见裂：淡金铭文膜覆盖现实，错误名字像锈迹。

抢笔：烧红铁丝般的笔画从锈层里抽出，带焦味和钢笔划纸声。

写骨：笔画落入船骨，船里响起低低旧名混响。

命名庭声音：钢笔刮纸、印章落下、纸页合拢。

未名舟声音：旧名低语、锅盖轻响、灯芯噼啪、木骨低鸣。

## 4. 角色私密锚点

角色要有小执念。

老周的乡愁面。  
苏晚的人话药柜。  
铁牙的断臂龙骨共振。  
云织找一扇缝不回去的门。  
陆小满的小铅笔。  
卿禾的错字本。  
沈砚的旧守灯人徽章。  

这些锚点要反复回收，不必每次都大，但每次出现都让读者更靠近人物一点。

## 5. 沈砚悲剧魅力

沈砚曾是守灯人。

他见过无名者因为没有合法身份而彻底消失，所以他变成一个极端的人：宁可给人一个错误的名字，也不愿让人没有名字。

他的错在于，他把“有名”看得比“自己决定叫什么”更重要。

卿宸要赢他，不是靠喊他坏，而是证明：被改成材料的名字不是救命，是另一种消失。

## 6. 执行硬口径

后续正文必须做到：

1. 每章命名能力至少有一种感官。
2. 每50章有情绪缓冲带。
3. 每30-50章回收角色私密锚点。
4. 沈砚每次出场至少展示一次“合法有效但残酷”的压迫。
5. 爽点后要有一点烟火气，烟火气后要有下一口危机。
