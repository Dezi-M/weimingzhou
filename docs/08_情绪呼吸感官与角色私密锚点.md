# 《未名舟》情绪呼吸、感官系统与角色私密锚点

## 1. 本轮自省

前一轮基建已经解决了长线骨架、敌方经济和收集追读，但仍有一层不足：温度。

如果只有正义一直爽，没有呼吸感，读者会兴奋但不一定留下。真正让人跟两千章的，是他想知道船上的人今晚吃什么、谁还没睡、谁把旧物藏在枕头下。

## 2. 情绪缓冲带

每50章安排1-2章低压日常或半日常章节。

低压章不拖节奏，它承担三件事：

1. 让船像家。
2. 让角色像人。
3. 让下一场大战有更疼的东西可失去。

可写内容：

- 老周研发新菜。
- 卿禾教孩子们认字。
- 苏晚整理药柜。
- 铁牙夜里听断臂和船骨共振。
- 云织缝一扇永远缝不回去的门。
- 陆小满记下强者说得太满的话。
- 卿宸在船头听旧名低语。

## 3. 元视界感官

命名不是抽象规则，而是一套读者能看见、听见、闻到的异能体验。

见裂：淡金铭文膜覆盖现实，错误名字像锈迹。

抢笔：烧红铁丝般的笔画从锈层里抽出，带焦味和钢笔划纸声。

写骨：笔画落入船骨，船里响起低低旧名混响。

命名庭声音：钢笔刮纸、印章落下、纸页合拢。

未名舟声音：旧名低语、锅盖轻响、灯芯噼啪、木骨低鸣。

### 3.1 命名感官分级

命名感官是为了让规则落到身体上，不是为了每次都放烟花。按场面大小调强度：

| 命名规模 | 推荐感官强度 | 使用手感 |
|---|---|---|
| 大场面：抢回城市名、打穿命名庭、卷末高潮 | 全感官饱和，视觉、听觉、触觉、嗅觉都可以上 | 让读者知道世界被改写了。 |
| 中等场面：抢回伙伴真名、对抗巡听灯、守住一条船规 | 2 种感官即可 | 例如灯膜和纸响，或焦味和刺痛。 |
| 小规模：抢回普通人一笔、打断定名钉、挡一次白线 | 1 种轻感官 | 例如指尖一烫、纸边轻响、钉身一凉。 |
| 日常回忆：提及旧名、摸到旧物、想起旧称呼 | 可以不用能力感官 | 靠旧校牌、碗底灰、铅笔头、门线结本身说话。 |

如果读者已经知道这一类能力怎么疼，后文可以少写一点，让具体选择和旧物变化顶上来。

## 4. 角色私密锚点

角色要有小执念。

老周的乡愁面。  
苏晚的人话药柜。  
铁牙的断臂龙骨共振。  
云织找一扇缝不回去的门。  
陆小满的小铅笔。  
卿禾的错字本。  
沈砚的旧守灯人徽章。  

这些锚点要反复回收，不必每次都大，但每次出现都让读者更靠近人物一点。

## 5. 沈砚悲剧魅力

沈砚曾是守灯人。

他见过无名者因为没有合法身份而彻底消失，所以他变成一个极端的人：宁可给人一个错误的名字，也不愿让人没有名字。

他的错在于，他把“有名”看得比“自己决定叫什么”更重要。

卿宸要赢他，不是靠喊他坏，而是证明：被改成材料的名字不是救命，是另一种消失。

## 6. 执行软口径

后续正文优先做到：

1. 涉及命名能力时，按规模选择感官强度；小规模可只留一种轻感官，日常回忆可不用能力感官。
2. 每50章尽量有情绪缓冲带；也可以用半页温情、一个笑点或一件旧物的新伤痕代替整章低压。
3. 每30-50章尽量回收角色私密锚点。
4. 沈砚出场时优先展示一次“合法有效但残酷”的压迫。
5. 爽点后要有一点烟火气，烟火气后要有下一口危机。
