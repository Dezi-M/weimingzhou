# 《未名舟》宏大架构：舰队、陆地与命名庭战争

## 1. 总升级口径

《未名舟》不能长期停留在“一条船逃命”。

真正的长线应该是：

单船活命 -> 船团救人 -> 舰队开战 -> 拼出陆地 -> 万界自命。

船是第一口气。  
舰队是第一支军团。  
陆地是第一份文明资格。  
万界自命，是这本书最终要打出来的法。

宏大感不能凭空拔高，要从小东西里长出来：一口锅、一卷绷带、一盏灯、一扇门、一块求救木牌、一条老街。读者先相信这些东西能救一个人，后面才会相信它们能托起一支舰队、拼出一块陆地。

## 2. 四条宏大成长线

### 2.1 船体线：无名船骨 -> 未名舟 -> 未名旗舰

未名舟早期是一条非法救命船，中期成为承约平台，后期成为万界拒名舰队的旗舰。

它不能变成皇帝宝座，也不能变成“主角一个人的神器”。它越强，越要暴露一个事实：这条船是由被救回来的普通人共同长出来的。

可见升级：

- 船骨：只够救七十人。
- 船舱：灶火、医舱、兵骨、市井、承约墙。
- 甲板：军团甲板、空白甲板、万界承约甲板。
- 旗舰：不命令万船改名，只为万船保留拒名和退约的空白位置。

### 2.2 船团线：单船 -> 拒名船团 -> 未名舰队

后续要出现别的拒名船，不能让未名舟永远单打独斗。

第一批拒名船不该像正规战舰，而应该像不同世界的残命：

- 被废名港口拖出来的破驳船。
- 由医院走廊和病床拼成的回声医舟。
- 由旧粮仓、锅炉和灶台拼成的灶火补给舰。
- 由桥梁、门框、铁轨拼成的守线索舰。
- 由校车、钟楼和儿童灯架拼成的童灯巡舟。
- 由市场棚、钥匙铺和旧秤拼成的市井驳群。

它们不是未名舟的附庸，而是把自己的求救木牌挂上承约墙后，选择一起开战的船。

### 2.3 军团线：伙伴职能 -> 职能队 -> 舰队兵种

每个主要伙伴都要长成一个可被读者记住的军团方向：

- 老周：灶火队 -> 灶火舰队，管粮、火、士气和烟火反制。
- 苏晚：回声医队 -> 回声医舟群，管救伤、欠命账、删债反击。
- 铁牙：守线队 -> 兵骨索舰，管登船、拖舰、防线和近战。
- 云织：缝城队 -> 缝界工程舰，管临时岸、阵地、残城拼接。
- 卿禾：归家灯队 -> 童灯巡舟，管黑航道、召回旧声和高代价探路。
- 陆小满：追问席 -> 承约调度舰，管公开规则、退约、分账和问责。
- 迟灯：内线队 -> 偷价斥候舰，管市场渗透、假背叛和敌方账本反挂。

军团不能只换名字。每次升级都要改变战斗方式，让读者看见“这支队伍能做别人做不到的事”。

### 2.4 陆地线：碎街 -> 临时岸 -> 未名岸 -> 自命大陆

陆地要出现，而且要作为中后期最大爽点之一。

未名舟初期救的是人，中期救的是城，后期救的是“人还能不能重新生活在一块不被命名庭批准的地上”。

陆地成长阶段：

1. 碎街：菜市场、学校门、医院走廊、钟楼、桥洞漂在黑航道里。
2. 临时岸：云织用缝线和拖城索把碎街拼成能站一夜的岸。
3. 未名岸：第一块不会沉的陆地出现，命名庭宣布非法，船民照样在岸上开灶、换药、上课、修门。
4. 自命大陆：多个世界碎片和自命印互相承认，形成不靠命名庭登记也能存在的陆基文明。

陆地不是安全屋。陆地出现后，敌人会更狠，因为船可以逃，陆地要守。

## 3. 拒名舰队构成

### 3.1 未名旗舰

核心：未名舟本体。

作用：掌舵、承约、调度、抢笔、写骨、保留空白甲板。

限制：不能替别的船命名，不能把所有船变成自己的舱室。舰队越大，越要允许各船保留自己的怪脾气、旧规矩和退约权。

### 3.2 灶火补给舰

来源：旧粮仓、灶台、饭馆、锅炉房、夜市。

战斗方式：用饭香、锅烟、灶火和粮线对抗敌方饥饿诱惑。它不只是后勤，还是士气武器。

名场面方向：白灯舰队围剿时，未名舰队全灭灯，只剩灶火一锅锅亮着，船民靠饭香识别自己人。

### 3.3 回声医舟

来源：急救方舱、病床、药柜、旧袖章、未结案伤情牌。

战斗方式：救伤、记债、反删债。敌人宣布“已尽力”时，医舟让每一笔未救之债重新出声。

名场面方向：命名庭删债医署压境，苏晚让整片黑航道响起床架声，所有被结案的人一起敲两下。

### 3.4 兵骨索舰

来源：老兵、守线者、桥梁、铁轨、断臂骨、登船索。

战斗方式：拖舰、登船、防线、反冲锋。它不是把人重新压成兵料，而是让弱者身前永远有一条线。

名场面方向：敌方白舰撞来，铁牙把断臂骨茬往索上一扣，说“线在，人不退”。

### 3.5 市井驳群

来源：菜市场、钥匙铺、修鞋摊、小饭馆、旧秤、门帘。

战斗方式：补给、藏人、情报、换路、拆钉。市井驳群不威风，但最难被命名庭一次性清空。

名场面方向：敌人搜不到未名舟，因为整条市井驳群把入口改成“买菜绕路”，追名兵在摊位里迷路。

### 3.6 童灯巡舟

来源：童灯队、校牌、旧课桌、校车、灯架。

战斗方式：探黑航道、召回旧声、照见敌方假安全航标。

限制：每次点灯都要有伦理代价，不能把孩子写成永动能源。卿禾的成长不是“点更亮”，而是学会什么时候不点、让谁一起决定。

### 3.7 缝界工程舰

来源：云织、旧门、桥、布棚、城市缝线、断路。

战斗方式：临时阵地、修船、拼岸、缝城。它是陆地线的核心。

名场面方向：一场大战里，云织不是修船，而是在炮火中把三座残城缝成一块能站人的岸。

### 3.8 承约调度舰

来源：承约墙、追问席、陆小满的铅笔、分账锅、退约牌。

战斗方式：公开任务、调队、分账、退约、反暗约。它保证舰队不是新命名庭。

名场面方向：敌人说“没有王调不动万船”，陆小满敲承约钟，让一百块木牌自己亮出愿意承担的那一段。

## 4. 陆地怎么写

### 4.1 陆地不是地图奖励

陆地出现，要同时带来三件事：

1. 生活：能开灶、睡觉、换药、上课、修门。
2. 战争：不能像船一样逃，要有人守。
3. 法理：命名庭不承认它，它却实实在在站住了。

如果陆地只是一块更大的船，那就失败。陆地要让读者感觉“这群人终于有地方把碗放稳了”。

### 4.2 第一块陆地建议

第一块陆地不要叫大陆，先叫“未名岸”。

材料：

- 江城钟楼半截。
- 菜市场青石门槛。
- 学校北门。
- 急救方舱地板。
- 旧桥一段。
- 一百块求救木牌背面刮下的木屑。
- 地球第一枚自命印的一点红泥。

形成条件：

最好由不同人开口，不是卿宸一个人写成。

老人说要有灶。  
苏晚说要有药柜。  
铁牙说要有守线桩。  
卿禾说要有一张不用点灯也能看清的课桌。  
陆小满说要有退岸权。  
云织最后缝线。

章末名场面：

命名庭宣布：非法陆基，不予承认。  
老周把锅往青石上一搁，说：“非法也先把饭吃了。”

### 4.3 陆地线的长期代价

陆地会吸引更多人，也会制造更大的取舍：

- 船能救快死的人，陆地要养活留下来的人。
- 船可以跑，陆地要守。
- 舰队可以拆开，陆地会让人产生归属、争吵、产权和新规则。
- 命名庭会诱导陆地居民要求“合法地名”，把未名岸改成附属港。

陆地越宏大，越要补日常。没有日常，陆地只是概念。

## 5. 命名庭战争体系升级

命名庭后期不能只靠追名兵和沈砚。要成长为能压住舰队和陆地的庞大体系。

### 5.1 合法港群

卖水、药、床位、身份和安全窗口。

打法：用真救援逼未名舟承认合法名。

### 5.2 归档舰队

专门收回拒名船、废名港、未登记陆基。

打法：不一定杀人，而是把人重新写成岗位、材料和归档资产。

### 5.3 删债医署

负责把死亡、亏欠、未救者清成“已尽力”。

打法：消灭记忆和债务，让受害者连追问资格都没有。

### 5.4 童灯营

命名庭培养的儿童探灯系统。

打法：用“孩子能救更多人”绑架未名舟伦理，让卿禾和童灯巡舟不断面对代价。

### 5.5 万名库

储存万界被改过、买过、卖过、赐过、删过的名字。

打法：不是战场敌人，而是最终证据库。打进万名库，等于让万界第一次看见自己曾经被怎样改写。

### 5.6 初名法庭

审判一个人、一座城、一个世界有没有资格自命。

打法：它不问你想叫什么，它问“谁承认你”。未名舟最终要打穿的不是一艘舰，而是这套承认机制。

## 6. 卷级名场面池

1. 黑航道深处，第一艘陌生拒名残舟亮起灶火回应未名舟。
2. 命名庭白灯舰队围剿，未名舟不是单船逃走，而是拒名船团第一次分舰作战。
3. 灶火舰、回声医舟、兵骨索舰、市井驳群、童灯巡舟第一次组合成完整战阵。
4. 云织把旧医院、旧学校、旧菜市场缝成第一块未名岸。
5. 万船挂木牌，每块木牌都不是船名，而是“我愿意承担哪一段”。
6. 命名庭宣布未名岸非法，老周先开饭。
7. 初名法庭问“谁承认你们”，未名舰队和未名岸上的普通人同时回声。
8. 终局万界自命，未名舟仍不取正式名，把空白甲板留给后来者。

## 7. 防俗规则

宏大不是把单位变大，宏大是让小人物的选择变成世界后果。

更深一层：舰队和陆地不能只是反抗军资产，它们要证明“反终名”。

舰队证明：一条船不必成为唯一方舟。不同残舟可以挂约同行，也可以保留自己的旧规矩和退约权。

陆地证明：一个地方不必被命名庭承认，才算存在。未名岸可以同时是市场、病房、校门、守线阵地和回家路，不必被压成一个合法地名。

如果舰队最后只听卿宸一人调度，或者未名岸最后变成未名舟的附属基地，就违背本书大脑洞。

尽量避免：

- 只把“船”升级成“十万战舰”，却没有新规则。
- 只写主角一声令下，万船听命。
- 舰队像军队编制，丢掉饭、药、灯、门、木牌这些本书味道。
- 陆地变成普通基地，没有法理冲突和生活细节。
- 命名庭变成只会发兵的普通帝国。

正确写法：

- 每一艘船都有旧物来源、普通人来源和不可替代的功能。
- 每一次舰队调度都要体现承约、退约、分账和公开规则。
- 每一块陆地都要先能住人，再能打仗，最后才谈文明。
- 每一次敌方升级都要带来具体诱惑：水、药、床、合法身份、亲缘证明、世界承认。

## 8. 落进正文的执行提醒

031-050：先露“船外回应火”。不要立刻舰队化，只让读者知道黑航道里还有别的拒名残舟。  
051-080：第一次救下一艘小残舟，让它不归并未名舟，而是挂约同行。  
081-130：江城残街不只扩舱，要出现“能站一夜的临时岸”。  
131-200：地球自命印打开时，让第一块未名岸出现雏形，为第二卷船团和陆地线打底。  
第二卷：拒名船团成形，废港变临时港，临时港再变临时岸。  
第三卷：职业军团成型，混合舰队第一次正面打赢一场正规舰队战。  
第五卷：王庭招安时，敌人不是只招安未名舟，而是分别招安舰队和未名岸。  
第七卷：承约平台大战，证明没有王也能调动舰队。  
第九卷：归地球线，未名岸与复地球协议正面对撞。  
第十卷：万界自命，船、舰队、陆地一起完成终局。

每次写宏大场面前，先问一句：

这个大场面里，有没有一口锅、一条绷带、一块木牌、一个怕死但还是伸手的普通人？

没有，就还不够《未名舟》。

再问第二句：

这个大场面有没有打破某个“唯一正确答案”？

如果没有，它只是大战；有了，它才是《未名舟》的大战。
