# 《未名舟》伏笔索引

本索引只存未来会继续反咬正文的活伏笔。它不是材料仓；没有写在这里的内容，续写时默认不参与判断。

维护口径：

- 一条伏笔只写一句到三句，优先写物件、伤口、敌方样本、待反制。
- 已结算但仍能变成下一轮敌压的，标“已用，仍可反咬”。
- 新伏笔必须服务“终相猎场 vs 人间万象”“人间开阵”“奇锋 / 军团 / 临时岸”。

## 当前最活跃

- 038 章末无形脚印：已用，仍可反咬。039 升级为第一只猎足伸进黑水，证明它不是普通登船压迫，而是终相猎场第一次踩向未名舟。
- 第一只猎足：先踩影、再拴影、再换影。039 已确认它不急着杀人，先压扁人和旧物之间的关系，诱未名舟提前开阵或错用旧物。
- 039 猎足样本：猎足已记住钥声、碗声、校牌冷光、残舟帆影、船头木牌旧字和众人站出来时的犹豫，后续可被敌人反向利用。
- 卿宸掌心短脚印：039 为挡钥影被踩出；040 被卿宸压到船头木牌背面，配合“错一次”骗偏猎足落点一寸。脚印已加深并多灰黑边，后续也可能成为猎足锁定卿宸的伤口入口。
- 三丈线：残舟和未名舟互不吞并的距离，039-050 变成伏击线、并战线和不并名的第一条船约。
- 残舟帆影：船体不并船，但帆影可以先并战。040 章末黑线搭上帆影边，残舟被拖沉一寸；041 要处理拖帆危机和三丈线并战。
- 卿宸的第一下：每章必须有一个只有他能做的动作，挡终相第一压、看出另一面、错开猎足落点，或把选择权递回普通人。
- 卿禾旧校牌：039-050 不再只照名字，而是照回“人还不止一种用途”。代价是她会失去一段普通校园记忆。
- 邱成缺口碗：039 已在最渴时偏开碗口，不接黑水；后续倒扣时才能扣住影子做阵眼。失日债是以后这只碗可能再也盛不满水。
- 老周铁锅：039 锅灰已被压出一粒；040 锅灰描出敌足轮廓，老周用锅盖扣住半圈灰线，锅盖少一层饭香灰。锅底裂后，饭香会被敌人采样。
- 梁春枝断秤：能称影、称残舟不愿并船的距离，也能称出己方欠债。越准越疼，秤杆会继续崩角。
- 孙福来旧钥串：039 已按住钥串不开门，钥齿咬破掌心；040 面对门影里的假钥声，他把钥串握进伤口止声。钥声仍被猎足记住，后续会成为反诱饵。
- 旧绿门：现实里是市井舱入口，影子里可能被踩成敌方登船门。040 旧绿门影子外开一指后被暂封，门下留极细黑线；门缝战法和敌方登船风险都从这里生长。

## 039-050 待结算

- 猎足样本：040 已获得足印碎黑一片，反向钉眼已可见。它既是未名舟短暂预判落点的样本，也是猎足反向钓船的风险。
- 人间开阵第一次完整喊出：050 才完整打出“人间，开阵”。039-049 只用动作、半句、旧物应声和普通人选择逐步逼近。
- 残舟同行三十丈：050 结算不是收编残舟，而是不并船也并战。残舟要保留脸、脾气、退约权和自己的旧物。
- 失日债：锅裂、碗伤、校牌失忆、钥齿断、秤崩角、帆影变薄，胜利必须留下普通日子被迫上战场的损耗。
- 敌方反学习：猎足被打断后，不是失败退场，而是带走“人和旧物关系能开阵”的样本。

## 人物长线

- 卿宸：不是规则解释者，也不是赐能者。他的成长线是从挡第一下，到看多相，再到把多个普通人的反用途动作接成战场。
- 卿禾：不是牺牲记忆的工具。她要越来越主动地判断“该照谁、照哪一面、错照会害谁”。
- 老周：赤釜军种子。锅不是温情背景，是饭、火、烟、饥饿和战场遮蔽的核心器官。
- 邱成：悬衡线种子。他越心疼碗，倒扣碗那一下越狠。
- 梁春枝：悬衡奇锋核心苗。她的嘴和秤都要锋利，负责把敌人不承认的重量称出来。
- 孙福来：开阙奇锋核心苗。钥匙先露怯、人再硬顶，是他的活人味。
- 苏晚：051 后进入续脉线。她不能把伤者交给合法救援床，才会长出移动救援仓和白垒军后勤根基。
- 铁牙：断渡线种子。他负责把“不能被推去当兵料的人”挡在身后，不只是打手。

## 组织长线

- 七奇锋：开阙、返灯、悬衡、续脉、断渡、影钩、回昼。它们是针眼小队，负责破口、潜入、返照、抢气、断渡、钩敌和夺时。
- 六大战军：赤釜军、白垒军、裂桥军、归轨军、晨灯军、守岸军。它们是大战军团，负责饭火、救援、防线、撤离、照明和陆基防守。
- 奇锋与军团不是升级关系。奇锋开针眼，军团压战线；奇锋可以给军团开口，但不会长大成军团。
- 承约制：不是任务面板，而是人、物、船、外族之间保留退路的战斗协作方式。

## 后续伏笔

- 051-060 水药床战场：合法救援港给真水、真药、真床，但要求登记用途。未名舟要抢出移动救援仓，而不是只拒绝诱惑。
- 061-070 沈砚压制开阵：沈砚要定“地球旧物不得成兵”，逼卿宸从单点开相升级到多相接战。
- 071-080 双船并肩：残舟有自己的船脸、旧物、脾气和退约权。第一条船团约是不并名，只并力。
- 081-090 废名浅滩：门牌、床号、路牌、柜门和旧名残片堆成浅滩，未名舟第一次主动截押运线。
- 091-100 临时岸：学校门砖、菜市场青石、旧门、锅灰、绷带、断秤钉和校牌冷光共同缝岸。岸能站一夜，但白舰群会压来。
