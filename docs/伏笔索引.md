# 《未名舟》伏笔索引

本索引只存未来会继续反咬正文的活伏笔。它不是材料仓；没有写在这里的内容，续写时默认不参与判断。

维护口径：

- 一条伏笔只写一句到三句，优先写物件、伤口、敌方样本、待反制。
- 已结算但仍能变成下一轮敌压的，标“已用，仍可反咬”。
- 新伏笔必须服务“终相猎场 vs 人间万象”“人间开阵”“奇锋 / 军团 / 临时岸”。

## 当前最活跃

- 一寸成交金线：001 卿宸从“成交”二字之间扯下一寸金线，打断卿禾的成交锚。它是第一件敌方交易残物，也是命名庭追捕“非法夺货者”的证据。
- 卿宸掌心白钉伤：001 白钉成交锚扎进掌心。后续可成为命名庭锁定卿宸、追索非法夺货、反向标价未名舟的入口。
- 卿禾残灯纹：001 “燃灯童”未完全成形，童字少一钩，残灯纹贴过无名船骨。她不是旧版牺牲容器，而是第一位让船骨拒绝标价的人。
- 点名纸和课堂旧声：001 女老师点名，孩子们答到，课堂旧声干扰成交锚。点名纸上“卿禾”的禾字只剩一竖，贴进船骨空白纹路，成为第一份非货物见证。
- 无名船骨拒绝标价：001 黑水里出现无名船骨，先浮出“标价”，后因卿禾、点名纸和卿宸成交金线而停住。章末未编号物拒绝标价并被命名庭要求立刻捕获。
- 急救室成交旧债：001 江城急救中心三号抢救室已成交，女医生和病人仍在被带走。后续若抢急救资源，要记得这不是普通补给，而是第一场未救完的伤。
- 焦黑锅底灰：001 路边摊铁锅被写成低阶灶胆，锅底灰落到卿宸鞋面，章末随船骨靠拢。它可以自然长成灶火线，但不是保留旧锅义务，而是本章新生长的火种。
- 女老师和保安：001 女老师用点名册做见证，保安用正在变成护栏料的胳膊挡线。他们是第一批普通人主动选择，不是背景群众。
- 七口欠命：002 卿宸用白钉伤和成交金线在船骨上刻“欠”，救人从标价变成欠命。七道欠痕是第一块歪甲板的根，也是后续敌人追索“盗舟”的口子。
- 钉舟与盗舟论处：002 命名庭捕获未编号物失败后改派三艘钉舟，要求交还附带拖拽货物，否则按盗舟论处。003 必须接住这个当场追捕。
- 卿禾记忆缺口：002 卿禾用残灯纹和点名纸照手念名，转头忘记周安的名字。她不是被动牺牲，而是主动点灯后的第一笔代价。
- 周安铅笔头：002 周安用刻着“安”的铅笔头自己答到，补上卿禾忘掉的名字；003 写“周安不退”；006 划“不代”；008 写“锅边”；009 写“嘴有”时铅芯碎进木茬；010 用无芯木杆卡白牌，木杆折断。它是“自己答 / 不代答 / 自取饭 / 自己说话 / 拒恶名”的第一枚小锚。
- 老周裂锅柄：002 老周从焦黑锅底灰里浮出，用半截锅柄和锅灰挡捕获钉，锅柄被钉裂；008 用裂锅柄和锅灰逼出白柜锅边粥，手背被烫红；009 刮出饭盒湿底和锅灰抹进旧绷带，给非法饭线留下第一点饭味。它是灶火线自然生长出来的第一个活人入口。
- 梁朵饭盒与陈一桥篮球网：002 梁朵半只饭盒卡进船骨，陈一桥用篮球网先推小孩后救自己；007 梁朵重画丑狗，饭盒裂口与空白腕带断扣生成第一枚非法护手钩；008 饭盒接住第一点不登记锅边热粥；009 饭盒挡白泡沫，断网拖旧绷带饭线；010 半块白牌砸进饭盒，断网和饭线共同撕裂“饥舟”白牌。后续欠梁朵一个不漏的新饭盒，也欠陈一桥一张不断的网。
- 抢救灯影未还：002 女医生没能上船，只把抢救灯影推给船骨，口型是“记着”。急救室、病人和女医生仍是医线第一口未还债。
- 本人不退价牌：003 卿宸在价牌背面刻下“本人不退”，但不是替七口气答，而是让梁朵、陈一桥、周安等人自己答。它是第一份本人答复样本，也被命名庭记录成盗舟样本。
- 盗舟成立 / 买方认领：003 拒还成功后，命名庭判定盗舟成立并开放买方认领。三号抢救室买方已收到样本，准备申请认领未结案伤口。
- 白泡缝路：003 未名舟从三艘钉舟中间硬挤出去，得到第一条白泡缝路。它不是安全航道，而是通向门牌、病床号、课桌腿和路沿石沉浮的更黑水域。
- 卿禾第二处记忆缺口：003 卿禾用点名纸让七个孩子自己答名后，忘记梁朵是谁。梁朵用“狗画得很好的那个”把自己递回去。
- 点名纸最后一角裂开：003 白钉钉向欠字时，卿禾护住欠字，点名纸最后一角被钉裂。点名纸仍可作见证，但越来越少。
- 苏晚旧绷带：004 水底女医生姓名被卿禾照出为苏晚。她仍在给病人按压，只把一截旧绷带推上船；005 第一滴未登记盐水先湿了旧绷带和灰白医线，让水底按压声清楚半分。
- 灰白医线：004 医线能压住伤口、防止买方顺伤口认领，但不是治疗完成；005 医线磨漏盐水袋角、缠住半截床栏，开始从“压伤”向临时伤铺边延伸；006 压住罗琴腕口，证明未名舟可以给一口救援但不替人登记。卿宸掌心医线仍连着水底按压手，不能硬拉救人。
- 江城急救押运线：004 章末病床轮带来新认领单；005 未名舟抢到半饭盒未登记盐水、一角止血棉和半截床栏；006 急救车厢底盘贴舷清收失败一段，敌方获得“非法急救 / 代登失败 / 自答污染押运流程”样本，下一轮改用陪护项。
- 许愿的选择：004 许愿主动选择“我也要这个脏的”；005 第一滴盐水在他舌尖写出“临时伤者编号十七”，他主动吐掉会写字的水，又让第二滴盐水刮掉舌尖编号，仍选择不登记。
- 罗琴与十七条空白腕带：005 罗琴自己说“我叫罗琴，不是空床三号”，用只剩“琴”字的旧工牌暂缓重登；006 她拒绝代登、用断铅笔卡扣口、接住盐水，半身喘上未名舟伤铺边。她失去旧工牌，欠周安一根带橡皮的新铅笔，腿上仍有蓝钩勒痕。
- 伤铺边：005 半截床栏、裂床轮、锅灰、盐水和周安写下的“床坏”嵌进船骨，形成第一道低矮伤铺边；006 刻入“不代”；007 嵌上非法护手钩；008 写下“锅边”；009 写下“嘴有”；010 留下七十道不齐咬痕和半块白牌。它不是完整病床，只能让重伤者少蜷一口气，但已经开始变成续脉 / 白垒线的防护器官。
- 梁朵临时陪护项：006 梁朵差点替罗琴确认，后说“那我替我自己”；007 她用饭盒丑狗和个人手痕自己作答，打散合并陪护手套，但手背蓝线仍未消失。后续敌人可沿“临时陪护人”追踪她，也可诱她用登记换饭。
- 孟达：006 十七名未登记伤者之一，袖口有褪色外卖反光条。他用吊瓶管缠住蓝钩，为罗琴偏开半寸并自己说名；009 用吊瓶管和反光条自己钩住断网饭线，含到一点饭味；010 咬住吊瓶管参与七十口印，还把“我还欠客户一个差评没删”变成自己不只剩饥饿的见证。他仍在车厢担架里，是“自己取 / 车厢伤者 / 外卖旧业”三条线的活口。
- 空白腕带断扣 / 护手钩：006 第一枚断扣飞进未名舟；007 与梁朵饭盒裂口咬合成非法护手钩；008 护手钩挂住腕带和细管；009 锚住断网饭线但被拉歪；010 回弹刮裂“饥舟”白牌的舟字。它是敌方“代登失败 / 护手钩生成 / 自取进食规避 / 非法饭线 / 拒临时命名”样本，也可能反咬伤铺边。
- 陪护餐热粥 / 锅边热粥：007 急救车厢白柜推出真热粥；008 未名舟抢尚未装袋的锅边溢出损耗，罗琴、许愿、梁朵、老周、陈一桥各自自取第一口不登记热粥；009 锅边饭味被抹进旧绷带和断网，生成第一条非法饭线。后续敌人会围绕“自取进食”和“饭线供餐点”反制。
- 空碗配给缺口 / 饥舟白牌：008 章末白柜推出空碗；009 空碗统计嘴、采空腹反应、封口清算，失败后钉出白牌临时命名“饥舟”；010 白牌以七十口饥餐诱统一口印，被七十个不一样的齿痕咬裂。半块白牌留在伤铺边，是拒名证物，也是敌方继续追“非法供餐点存疑”的钩子。
- 非法饭线：009 新增伏笔；010 被白牌卡断后重新亮起，分出细岔连回旧绷带、护手钩、伤铺边和车厢吊瓶管。它不够喂饱人，但能让七十口气先挂住未名舟，后续可长成赤釜 / 白垒之间的早期后勤线，也会被敌方按非法供餐点追捕。
- 急救方舱接管：010 章末新钩。它给真药、真床、真命，但登记用途为“伤者”；危险在把活人压成只能被治疗和处置的用途。011 要抢床，不要只拒绝。

- 猎足样本：040 已获得足印碎黑一片，反向钉眼已可见；041 证明它会被猎足反钓；042 卿宸用它试出真假白点，并得到一抹空白脚泥；049 缺口碗扣住猎足半枚钉眼；050 半枚暗红钉眼嵌进碗底。足印碎黑不能当可靠雷达，空白脚泥不可碰，半枚钉眼会持续引来猎足 / 沈砚反制。

- 饭香灰样本：043 白桥偷走一粒饭香灰，票孔边缘浮出淡黄。后续敌人可用假饭香、锅烟定位或把锅压回供能炉来反制赤釜线。
- 账债样本：044 白桥没吃到人，却尝到“欠账”能被塞进门缝；045 它把欠账、饭香灰和钥声合成假救命门。后续敌人可把欠饭、欠命、欠救援变成开门、归并或诱导交人的口子。
- 冷白脚跟灰：045 旧绿门夹下半截白脚印后脚跟，梁春枝确认敌人的后脚跟也有重量。可作为猎足实体弱点样本，也可能被白桥反向污染断秤。
- 影子门 / 无脸人影：045 章末白桥学会反扣钥声，改敲旧校牌边缘的影子门，照出无脸人影说“我到”。046 卿禾识破“我到”伪装，照背影救回真实小孩影，但白桥拓下校牌空白，得到“背影 / 照错前影粉”样本，后续可伪造背影或换影误伤。
- 黑箭换影：046 章末不亮灯残舟主动开火，黑箭贴三丈线射向白影；白影退进冷光余痕，把影子换成未名舟船影。047 卿宸错开真船影、陆小满留半笔“认”、残舟断嗓女人咬断弩弦，黑箭前半截扎回白影胸口，黑血钉住三丈线。
- 双面影皮 / 旧物认错：047 白桥吞回冷白假影皮，得到未名舟 / 残舟双面影皮样本；048 它用影皮骗锅盖、旧钥齿、缺口碗、旧校牌认错对象，失败后带走“旧物认笨习惯、旧伤和代价”的样本，并把缺口碗推到阵眼边缘。
- 人间开阵第一次完整喊出：050 已完整打出“人间，开阵”。它不是口号，而是七十人站位、旧物应阵、残舟并战三十丈后才成立；猎场已刮走开阵样本，后续会反制。
- 残舟同行三十丈：050 已结算为“不并船，并战三十丈”，战后记成互欠。残舟保留脸、脾气、退约权和自己的旧物；这条互欠账后续可成为船团约雏形，也可被敌人诱导归并。
- 失日债：锅裂、碗伤、校牌失忆、钥齿断、秤崩角、帆影变薄、座位痕消失，胜利必须留下普通日子被迫上战场的损耗。
- 敌方反学习：猎足被打断后，不是失败退场，而是带走“人和旧物关系能开阵”的样本。046 白桥已从卿禾救影里学到背影样本和换影方向；047 从黑箭换影失败中得到双面影皮；048 旧物认错失败后确认可以逼缺口碗做阵眼；050 猎场退走前从锅烟、碗痕、秤星、钥血、校牌光、绷带湿痕、焦索铁锈和残舟帆影上各刮走一点。
- 合法救援港：050 章末浮出，给真水、真药、真床，但要求登记伤者。危险不在资源虚假，而在把活人压成“伤者用途”；051 要从苏晚、旧绷带和伤者本人回应切入。

## 人物长线

- 卿宸：不是规则解释者，也不是赐能者。他的成长线是从挡第一下，到看多相，再到把多个普通人的反用途动作接成战场。
- 卿禾：不是牺牲记忆的工具。她要越来越主动地判断“该照谁、照哪一面、错照会害谁”。046 已完成一次主动判断：不照脸、不答到，照背影并把人送回自己的残舟。
- 老周：赤釜军种子。锅不是温情背景，是饭、火、烟、饥饿和战场遮蔽的核心器官。
- 邱成：悬衡线种子。他越心疼碗，倒扣碗那一下越狠。
- 梁春枝：悬衡奇锋核心苗。她的嘴和秤都要锋利，负责把敌人不承认的重量称出来。
- 孙福来：开阙奇锋核心苗。钥匙先露怯、人再硬顶，是他的活人味。
- 苏晚：004 已通过水底抢救灯影和旧绷带入医线。她仍未上船，当前核心不是被救，而是“别停”：不能让伤口和抢救动作被买方结案。
- 铁牙：断渡线种子。他负责把“不能被推去当兵料的人”挡在身后，不只是打手。

## 组织长线

- 七奇锋：开阙、返灯、悬衡、续脉、断渡、影钩、回昼。它们是针眼小队，负责破口、潜入、返照、抢气、断渡、钩敌和夺时。
- 六大战军：赤釜军、白垒军、裂桥军、归轨军、晨灯军、守岸军。它们是大战军团，负责饭火、救援、防线、撤离、照明和陆基防守。
- 奇锋与军团不是升级关系。奇锋开针眼，军团压战线；奇锋可以给军团开口，但不会长大成军团。
- 承约制：不是任务面板，而是人、物、船、外族之间保留退路的战斗协作方式。
