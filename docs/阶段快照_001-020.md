# 阶段快照 001-020：开船、饭、医舱与黑航道

本快照只用于快速回忆，不替代正文、审计和当前状态账本。

## 阶段主菜

001-005 完成开篇承诺：地球被拆成货，卿禾被写成燃灯童，卿宸抢回“禾”的一竖，无名船骨醒，七十人上船，沈砚以真实救援和合法定名第一次压场，卿禾主动点亮归家灯。

006-010 把七十人落成七十张嘴：老周、米袋、锅、燃灯童登记牌和第一顿非法粥成立。饭香引来白舟追踪，半声“到”和同锅灶灰让未名舟成为拒名船，黑航道半开。

011-015 抢回急救方舱左前段和苏晚，立住回声医舱、伤情牌、救命债不能删、唐守义声痕、陈桂兰腕带、兵料铁牙和反挂钉。

016-020 进入压舱者、保命书、童灯队和黑航道第一战。顾槐自愿压舱，江柏伪保命书被撕，卿禾照破假安全航标，童灯队三席成为旧痛，卿宸第一次真正掌舵冲过黑航道第一段。
