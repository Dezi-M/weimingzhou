# 《未名舟》当前故事状态账本

本文件只记当前可续写状态，不替代总纲、阶段补纲或正文审计。它的用途是让“继续”时不用每次从整套文档里重新捞记忆。

更新口径：只写已经在正文成立的事实、短期牵引、旧账折痕和下一章手感。不要写写作课，不要写硬门禁。

## 账本保洁规则

本账本只承担“下一次继续时能接上”的工作，不承担全书档案馆工作。

- 三层窗口：近 20 章是高亮续写层，保留细节、伤口、敌方样本和章末钩；近 100 章是日常账本层，保留一句现状和唤醒条件；100 章外无变化但未来仍可能回收的旧账，迁入 `docs/15_长期旧账索引.md`。
- 50章定位：50章是阶段审计和轻保洁点，不是旧账迁出的硬线。它适合压短、标记、结算，不急着把仍有热度的旧线折走。
- 结算标记：旧账已经在正文中回收、牺牲、失效或转化时，先标 `[已结]` 或 `[已转]`，下次十章复盘时从日常账本移走。
- 定期清理：每 10 章顺手整理一次；每 50 章轻保洁一次；每 100 章再把 100 章外且长期无变化的旧账折到长期索引里。
- 轻口径：不要为了“完整”把所有历史都塞回来。若某条旧账暂时不会牵引下一章，让它安静待在长期索引里。

- 054：白票把小残舟船舷糊成“已自愿夜航”的票皮，脏布血字“退不了”被往船舷里压。未名舟不硬撕票，借白票自己写过的“退票需至桥尾”，让陆小满写“桥尾见退 / 票不代船答”。邱成用缺口碗向裂票下方问退，里面的人用额头撞船舷回应；苏晚用旧绷带把碗声接到退约牌下方，只认伤不认人；陈远撕下错题纸毛边卡住白票缝，老周锅烟、卿禾归家灯、孙福来旧钥三敲和退约牌一起撑出半只碗口大的退票口。白纸录下“退票口样本 / 错题纸边污染 / 碗印接收 / 烟灯缝路待仿”。退票口里推出一截断掉的剪票钳，脏布血字提示“补票的在船上”。
- 055：剪票钳试图把退票口往里“补”，并剪向碗印、退约牌和小残舟里那只手。白纸把那只手写成“补票员”，陈远认出这和称谓抵押一样不是本人写的。梁春枝用断秤逼白纸亮出“桥资：已付”却无数字，陈远拿错题纸白蜡边让钳口偏向欠账，苏晚用旧绷带让“补票员”的“补”字掉半。陆小满写“手不听，问手 / 要退，自己敲钳”，里面那只手撞钳柄，使剪票钳剪破自己的白票，露出钉着三枚断钉的烧黑木牌，中间一钉刻“不补”。断钳卡住“不补”钉，白纸记录“不补钉样本 / 断钳证物 / 抗补手待捕”。章末白灯深处更多崭新剪票钳影醒来。
- 056：批量补票钳影落下，不碰人和物，只剪白灯下的影：先剪陈远母亲喊“小远”的尾音，再剪退约牌“退”字影，又剪问路孔黄灰影，三十丈外药味一度断掉。卿宸坚持不把断钳证物和不补钉拿进船，慢半拍里退约牌影又被剪一口；卿禾不照字，只照影口。孙福来指出影从灯下走，老周用锅烟让钳影投出自己的影，梁春枝以断秤追问灯账，邱成缺口碗影护住陈远练习册红字里的“远”。陆小满写“灯下有影 / 剪影先问灯”“证在水上 / 影不登船”，第一排钳影被白灯账拖住，陈远母亲重新喊出“小远”。章末第二排白灯转向卿宸影子，白纸写“船主影未补 / 可先剪掌舵手”。
- 057：白灯剪中卿宸掌舵手影，卿宸右手短暂失去木牌触感，船头向白灯偏了一寸，白纸试图把掌舵手影补成船主名下。邱成站出来认回旧句“急的时候掌舵的先掌”，缺口碗把这句话接成见证；铁牙用索影挡嘴影，云织用旧线和血缠退约牌背面，梁春枝用断秤称“这一息”，陈远用裂开的练习册页护住陆小满笔影。陆小满写“掌舵先掌 / 事后记账 / 船不归手”，后又改成“不并名 / 只并力”。卿宸没有抢回独舵，只用掌心血把旧物影之间被剪出的白缝染脏，众旧物合力让钳影打滑。船头偏回半寸，但白纸记录“掌舵手影残 / 急掌语源已录 / 多物同舵待拆分”。章末裂缝旁多出一枚没剪成的空手印，远处白纸写“代舵席待开”。
- 058：白纸远挂“代舵席待开”不立刻进攻。苏晚给卿宸包右手，确认手指仍慢；卿禾看舵一息，云织补退约牌裂缝，问路孔黄灰低亮未灭，陈远撕练习册毛边给陆小满写“看舵一息 / 不坐席”。邱成用缺口碗只看左边一息，铁牙索、断秤和旧钥各稳一点。老周用碎面、锅灰、半粒盐做出一锅不像饭的热食，船上短暂笑了一次；白纸学习“饭”和“休息”，推出白凳白碗：“代看一息，供热饭一碗 / 食者，先坐”。
- 059：白碗饭香过水，诱导交出半粒盐的孩子“自认食者”，并把闻香、想吃、坐下都往代舵席登记上扯。老周没有骂孩子饿，而是让他说想吃，再把他摁回湿船板；陆小满写“坐船板 / 不坐白席”“闻香不记食 / 想吃不签收 / 坐处各自，不成代席”。孙福来、邱成、伤者和孩子们陆续坐回自己的船板坐痕；孩子用锅铲敲裂锅底糊痕，说“我想吃饭，不想坐它”。白碗露出空汤，白凳被烫出锅灰黑点，白纸记录“食者登记失败 / 代席未成 / 锅底坐痕已录”，章末写“席不必白 / 船板亦可量”。
- 060：白尺直接量船板坐痕，把孩子坐痕写成“食者轻舵”，缺口碗印写成“左重”，钥坑写成“锁位”，药痕写成“伤席”，血线写成“边”，并试图合成“合席代舵，可省一手”。孩子说“我想吃饭，不想坐它”，老周锅底糊痕、梁春枝断秤、孙福来旧钥、铁牙血线、云织退约牌和邱成缺口碗共同让“各痕可问 / 各痕可退”成立。白尺断一小截，卿宸右手慢慢动回一点；白纸记录“合席代舵失败 / 各痕可退 / 锅底坐痕污染尺样 / 退约牌可拆席 / 船板待钉”。章末白钉影浮在船板湿坐痕之间，钉尖对准船骨缝。
- 061：白钉以“有痕之板，可钉 / 无答之板，由主代答”逼卿宸成为船板代答者。卿宸右手接第一下钉痛，但让陆小满记成“挡钉一息 / 不作代答”。孩子用锅铲柄认回自己的坐痕，邱成扣回缺口碗并承认怕裂，孙福来横放旧钥不开船骨，苏晚用脏绷带按住药痕，铁牙用血线弹歪钉尖。卿宸扣下钉帽空白，不写答案，只在木牌上烫出五个“问处”；船板下长出五枚浅凹槽，白钉合钉失败并裂成两半。白纸记录“取船骨旧名，可补钉帽”，回应火残舟三下钉响，卿禾判断那边也被取过旧名。
