# 阶段快照 041-060：拒灯船群、白灯夜航与一口脏水

本快照只用于快速回忆，不替代正文、审计和当前状态账本。

## 阶段主菜

041-050 接住不亮灯残舟保卫战。群粮箱、修船钉、听源白税、小白耳、刀铺蓝门、客钩、同犯清单和白榜收网轮番压船；未名舟没有并船，而是让不亮灯残舟同行三十丈、可退、不归并。第二道回应火只稳住一息，后续以贴票残舟露面。

051-060 完成白灯夜航和主动抢水药。退约牌给出真水真药诱惑；白灯夜航队带来水柜、药柜、白床和陪护灯。马三娘、不亮灯残舟、贴票残舟和未名舟共同形成三船错拍：不亮灯遮影，贴票乱声，未名舟抢柜。最终抢到一口不登记的井纹脏水，孩子多撑一夜，沈砚改查分水。
