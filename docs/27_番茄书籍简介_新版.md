# 《未名舟》番茄书籍简介新版

本文件用于番茄后台书籍简介、封面副标、推荐语和后续宣传口径。当前主打不是“规则怪谈/解题文”，而是：

> 地球被拆成货，卿宸开船去抢回家。

## 推荐使用：后台主简介

地球文明审定完成。

新名：舟材世界。

医院连着病床被吊上天，学校门口的孩子被写成灯芯，活人、城市、职业、亲缘，全被万界命名庭标价成材料。

卿宸赶到江城一中时，妹妹卿禾手腕上浮出三个字：燃灯童。

他从那三个字里抢回一竖“禾”，也抢醒了一截没有名字的船骨。

从此黑水之上，多了一条破船。

船上缺水、缺粮、缺药，锅是漏的，门是旧的，绷带是脏的。可就是这些破东西，能挡白灯、抢医舱、拖老街、救残舟。

命名庭给安全身份、干净水、合法床位，只要你承认自己该被写成某种用途。

未名舟偏不认。

别人卖地球做船材，他偏要把船材抢回人间。

## 短版简介

地球被万界命名庭改名为“舟材世界”，人、城市、职业和亲缘都成了可拆材料。

卿宸从妹妹“燃灯童”的新名里抢回一竖“禾”，也抢醒了一条没有名字的破船。

从此，他带着一群被写成材料的人，在黑水末世里抢水、抢药、抢旧物、抢城市碎片。

命名庭给合法活路，未名舟偏要抢一条不被写死的路。

## 偏燃版

高处的人说：地球已审定，万物皆为舟材。

医院被吊走，校门变灯架，活人被写成货物。

卿宸只回了一句：我不认。

他抢回妹妹名字里的一竖，抢醒无名船骨，带七十个无票者冲进黑水。

锅能挡灯，药箱能救命，断秤能压账，旧门能撞开航路。

命名庭卖干净水，未名舟抢脏水喝。

命名庭给合法床，未名舟拖急救舱走。

命名庭要把地球卖成船材，未名舟就一块一块抢回来。

## 偏情绪版

卿禾不叫燃灯童。

老周不叫伙料。

苏晚不叫药箱。

铁牙不叫兵料。

当万界命名庭把人写成最适合的用途，卿宸开着一条没有名字的船，把他们从名册里抢回来。

船上什么都缺，只有旧锅、旧门、断秤、脏绷带和一群不肯认命的人。

他们不一定能赢得漂亮。

但每抢回一口饭、一滴水、一截旧街、一艘残舟，世界就多一点还能改变的余地。

## 一句话推荐语

- 别人卖地球做船材，他偏要把船材抢回人间。
- 她叫卿禾，不叫燃灯童。
- 高处的人说你是材料，他开船把你抢回人间。
- 一条没有名字的破船，抢水、抢药、抢旧物，也抢回人还能改变的权利。

## 推荐期封面口径

当前推荐期封面优先使用 `assets/covers/weimingzhou_cover_recommend_title_20260623.png`。

小图第一眼只抓三件事：

- 书名必须读得出。
- 船不是单船，要有舰队聚拢感。
- 天上拆城、红水冲船，要让读者先感到“地球被拆成货，主角开船抢回家”。

副标题不建议放进封面小图。番茄列表里小字容易变噪点，简介和推荐语负责解释，封面负责让人停手。

当前标题处理采用上方弧形日光金字：不加木板、破帆、红条或半透明底，只保留端正书名、暗描边和轻微破云暖光，让书名像从暗云里亮出来，避免横排大字压画面、裁底和 UI 贴条感。

## 简介避雷

- 不要开头就写“命名权规则是什么”。读者先看地球被拆、人被改、妹妹被写成灯芯。
- 不要把卖点写成“主角带领众人对抗命运”这类泛句。
- 不要过度强调“规则怪谈”。当前书型更适合“末世生存 + 旧物成战 + 船团成长”。
- 不要把未名舟写成普通救世方舟。它的辨识度是“不取名、不归并、不把人写成最终用途”。
