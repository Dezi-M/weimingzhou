# 阶段快照 021-040：老街上船、市井舱与回应火

本快照只用于快速回忆，不替代正文、审计和当前状态账本。

## 阶段主菜

021-030 抢江城老街，不拆街。旧冷柜、两瓶水、合法港试供、许青刀定位钉、拖街索、青石先过人、十个空白指印、公议钉和第一盏摊灯共同把市井舱立起来。船规雏形是“船头空着，债挂下面”，不是公议船名。

031-040 回应火残舟入场并靠近。拒灯船群、悬赏榜、听源白税、耳卡、刀铺蓝门、外舷挂齿、同犯清单、白榜残网等压力陆续出现。未名舟开始学会与不并船的残舟互相照应，而不是把一切收编。

## 当前仍可唤醒的资产

- 江城老街：旧绿门、青石巷、菜市、旧钟楼、刀铺蓝门、面缸声、杂货铺门槛。
- 市井舱：未成但已有街口、摊位、断秤、钥匙、刀背、旧磨石、门牙等战斗物。
- 许青刀与许陈氏：救人不救铺子、腕底定位钉、公议钉、刀背可问刃。
- 孙福来：旧钥匙、暗钥投毒、不要被当钥匙。
- 梁春枝：断秤、骂人、称水称账。
- 邱成：缺口碗，动心但能被旧账压回。
- 云织：缝材旧名被松开，灰蓝旧棉线与旧门缝线是市井舱连接物。

## 未结旧债

- 许青刀定位钉仍未彻底拔除，只被反挂到账本下。
- 合法港已经确认市井舱坐标，沈砚能把“救人的规矩”反用。
- 外舷挂齿、白榜残网、听源样本、刀背白印都让敌方学过一轮。
- 不亮灯残舟和后续回应火只建立互照基础，还未形成小船团战法。
