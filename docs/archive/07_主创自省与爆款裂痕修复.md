# 《未名舟》主创自省与爆款裂痕修复

## 1. 本轮自省

此前基建偏强在概念、系统和长线，但仍有一个问题：它更像一套能跑的世界机器，还不够像一本让读者上头的小说。

读者先被情绪拉住，再被设定留下。后续不能只问“体系是否自洽”，还要问“读者此刻恨谁、疼谁、想看谁赢”。

## 2. 四条爆款裂痕

### 裂痕一：恨点偏冷

“地球被改名”很新，但偏概念。001章必须补公开羞辱场景，让命名庭的恶具体落地。

落地：驻地球官员当众把拒名者改成示材台，证明改名就是改用途。

### 裂痕二：主角太正确

卿宸不能只是正义规则扩张器。他要有读者能理解的性格缺陷：护妹过度。

落地：010-020章中，他因不让卿禾主动点灯，差点让全船撞入陷阱。这个缺陷带来成长，不削弱爽感。

### 裂痕三：敌人设局不够主动

沈砚不能只在未名舟行动后反制。他要提前利用未名舟规则设局。

落地：021-030章埋假裂名定位钉，逼未名舟升级出自愿陈述权。

### 裂痕四：普通人高光不足

未名舟不是主角一个人的船。普通人必须能改变战局。

落地：每50章至少一个普通船民关键选择，写进审计硬规则。

## 3. 后续执行口径

每章写完，不只问有没有爽点，还要问：

- 有没有恨点。
- 有没有疼点。
- 有没有可视化战利品。
- 有没有敌方主动设局。
- 有没有普通人不是背景。
- 有没有赢大局输小局的后劲。

## 4. 下一步优先级

第一优先：重写001章，加入公开羞辱场景和“禾”字可视化战利品。

第二优先：重排010-020章，加入卿宸护妹过度导致危险、卿禾主动选择救局。

第三优先：重排021-030章，加入沈砚假裂名定位钉和自愿陈述权。

第四优先：写正文时每章用战利品画面收束，不用抽象概念收束。
