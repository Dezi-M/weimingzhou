# 《未名舟》031-050阶段补纲：拒名船悬赏榜

## 1. 结论

不建议重写主控、大纲或前030章强剧情链。

当前问题不是方向错，而是031-050已经写到039，后面缺一张可直接续写的阶段收束图。原定目标“拒名船悬赏榜、船外第一道回应火、沈砚取得见裂样本、低压日常、敌人学会假裂”已经在031-039里成立，后续只需要顺着现有钩子，把040-050写成一个完整小循环。

这份补纲只做续航用，不替代总纲。

## 2. 已写到039的有效资产

- 031：拒名船悬赏榜出现，回应火首次亮起，半焦米成为船外联系物。
- 032：榜上有价落地，陈远动摇但没被卖掉，卿宸见裂样本被白榜取走。
- 033：乡愁面失败但有效，假裂疑似出现，回应火也识别过假裂。
- 034：试岸第一次成立，只能站一只碗；退岸权被轻轻埋下。
- 035：三日清理令压到试岸，邱成岸民争议成立，回应火参与护岸。
- 036：隔水三息成立，黑航道里出现锅声、碗声、木门声，船外不止一处影子。
- 037：听见者名单出现，巡听灯压场，半块饼作为新求救物浮上来。
- 038：许青刀割坏自己的求救木板，避免“娘”字被合并；半饼槽成立。
- 039：第二口债救上娘和小的，但真正掰饼的大孩子还被白蜡封在水里。

这些内容和最新版 skill 的方向一致：第一页有抓手，章末有物理钩，敌方经济持续变招，过渡章也带动作和余波。

## 3. 航向观察哨

039章末的小手钩子很强，040最好别转场，直接接会更有追读惯性。

回应火残舟先别急着露脸。让它继续用锅声、碗影、门响和一点灰红火星交流，会比直接开出一艘船更有黑航道神秘感。

巡听灯可以在040-043输一小局，但最好带走“听源样本”或“半饼求救格式”。这样050以后敌人能造假回应火、假求救声，反派升维就不只停在审计里。

044-046可以稍微喘气，但不是水章。修锅盖、补半饼槽、许青刀母子吵架、卿禾灯芯变细，都能让读者缓一口气，同时把下一场仗的刀磨出来。

050要赢大局输小局：守住试岸、救回大孩子或至少夺回他的声音，和船外回应火建立隔水承约；但敌人拿走听源/半饼/隔水承约样本，下一阶段可以反造“假回应火”。

## 4. 040-050小循环

### 040 白蜡封口

直接接039章末。大孩子被白蜡封嘴，手里举着更小一块饼。巡听灯要求确认“原求救者”身份，想把他写成诱饵而不是人。

本章动作核心：卿禾用归家灯偏开白光，苏晚用热汽护住孩子一口气，许青刀母亲不讲理地骂出第一句人话，卿宸不替孩子定名，只先抢回他能发出的一点声音。

章末钩：孩子嘴上的白蜡裂了一点，没喊名字，只喊出半个“够”。

### 041 半饼原主

孩子不是来求救自己的，他掰饼是为了让娘和小的先浮上来。听见者名单想把“先救别人”登记成“自愿押后”，逼未名舟承认他可以被暂押。

本章动作核心：全船用半饼槽记账，不把他的选择写成牺牲，也不把他当诱饵。陆小满写下“先救，不等于放弃自己”。老周把锅底最后一点焦香送到水面，给孩子留住呼吸。

章末钩：巡听灯第三盏第一次暗了一下，但白纸背面浮出新字：听源可采。

### 042 巡听灯失聪

巡听灯开始采集船上所有回应声，锅声、碗声、门声都会被它标成非法联络。未名舟若继续回应船外火，所有人都会上同犯清单。

本章动作核心：铁牙守线，云织缝门声，梁春枝用断秤压半饼槽，孙福来用钥匙乱响，老周故意让锅声变得像骂人。不是消灭声音，而是让每个声音都带着具体人和旧物，白纸无法归并成同一类证据。

章末钩：一盏巡听灯失聪坠水，灯芯里却飘出一小片冷白蜡：已采样。

### 043 听源白税

敌人把“听见”变成经济。凡回应、敲锅、喊娘、传火者，都要缴听源税；缴不起，就由合法港替你保管声音。

本章动作核心：船上有人动摇，因为缴税似乎能换水药。卿宸如果只说不缴，会有点悬；给出活路会更好：未名舟用半饼槽、试岸灰、锅盖白痕建立临时听源账，谁的声音归谁，谁也不替谁收。

章末钩：回应火那边传来一声很轻的门响，像有人隔水说：我们也欠。

### 044 补锅盖

半日常章。老周补锅盖，许青刀母亲骂儿子，苏晚清点伤者，卿禾灯芯细了一圈。船上第一次有机会把刚救上来的娘和孩子安顿下来。

本章动作核心：补锅盖不是生活散文，而是为下一场仗做盾。锅盖豁口里嵌进巡听灯碎蜡，能挡白光一息，但会把老周的灶火声暴露给敌人。

章末钩：锅盖补好后，里面映出一条不属于未名舟的门槛影。

### 045 许青刀的刀背

许青刀想把割坏的木板修回去，他娘舍不得，觉得那是救人留下的疤。母子吵得很难听，但这场吵架能让许青刀从“被救者”变成能主动守木牌的人。

本章动作核心：许青刀不用刀刃，用刀背把半饼槽敲深一点，承认第二口债不是他的旧账，却愿意替那孩子守一晚。卿宸不夸他，只把木牌位置让给他。

章末钩：半饼槽下方多出一道小字：守槽者，许青刀。

### 046 不亮灯的船

回应火背后的残舟仍不露脸。它不敢亮灯，只把一扇烧坏的门影推过来，门后有很多人，但没有一个完整名字。

本章动作核心：未名舟先别贸然并船，也别急着隔水喊话。双方先用旧物试探：锅盖、门槛、半饼、试岸灰。陆小满提出“挂约不过船”：可以同行一息，不归并，不登记岸民。

章末钩：那扇门影轻轻合上，门缝里掉出一枚烧黑船钉。

### 047 同犯清单

白榜回收巡听灯残片，把所有敲过锅、挡过光、守过岸、听见娘的人列成同犯清单。合法港给出新诱惑：交出船外回应火坐标，可免除全船同犯登记。

本章动作核心：敌人小胜，船内真实害怕。这里如果写成大家整齐支持，会少一层人味；有人怕、有孩子想喝水、有伤者想上港，反而更真。卿宸也不替全船热血，只把代价摊开，让每个人看见交坐标会换来什么。

章末钩：邱成抱着那只站过岸的碗，先把碗扣在同犯清单上，说“我先不同。”

### 048 挂约不过船

普通人高光章。邱成、梁春枝、许青刀、孙福来这些非能力者共同把隔水承约做出来：不露脸、不并船、不上岸，只在三息里替对方挡一次白光。

本章动作核心：卿宸负责看裂，陆小满写账，真正改变战局的是普通人递出的碗、秤、钥匙、木板。回应火残舟回递一枚烧黑船钉，证明那边也愿意守同一条约。

章末钩：白榜没有抓到坐标，却抓到一行新词：隔水承约。

### 049 白榜收网

沈砚不一定亲身登场，可以通过白榜文书和假裂样本压场。敌人用卿宸见裂样本造出第一道假回应火，诱导未名舟偏航，试图让真正回应火暴露坐标。

本章动作核心：假回应火比真火更亮、更像救援。卿禾差点被它骗，老周闻出没有锅灰味，铁牙听出没有门响，许青刀看出半饼槽没有热。主角团靠前九章旧物识破假火。

章末钩：真回应火没有亮，只从黑水下敲出一下很低的船骨声。

### 050 第二道回应火

阶段大战与结算章。未名舟守住试岸，救回大孩子的声音，和船外残舟完成第一次隔水承约。残舟不归并，只挂约同行一段黑航道。

本章动作核心：所有资产回收一次：半饼槽守人，试岸灰挡白线，补锅盖挡巡听灯，许青刀木板守第二口债，隔水承约让回应火残舟不用暴露全貌也能参与反击。

赢的大局：第二道回应火真正亮起，证明黑航道不只有未名舟一条拒名船。

输的小局：白榜带走“隔水承约”和“听源账”的样本。下一阶段敌人可以制造假求救、假回应火、假挂约。

章末钩：黑水远处又响起第三种声音，不是锅，不是碗，也不是门，是一枚船钉敲在空木头上。
