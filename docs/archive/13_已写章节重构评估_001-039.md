# 《未名舟》已写章节重构评估：001-039

## 1. 总判断

不建议推倒重写，也不建议从0重新创作。

当前001-039的骨架是成立的：开局概念够新，命名庭的恶可见，未名舟的资产成长连续，十章节点有结算，031-039已经明显进入“旧物牵引、普通人选择、敌方经济变招、章末物理钩”的新版手感。

真正需要处理的不是重创，而是前20章的入口导线。前20章信息量、资产量、规则量都偏密，读者可能单章能懂，但连续读/听时不一定知道当下最该挂心的是哪一条明线。

2026-06-17已做入口质量加固：001-004补强“糖水/卿禾/七十人/童灯队旧账”的情感回声；006、008补强饭与童灯队债的连接；009-011清掉“上一章/第二章”等章节外指代，避免读感出戏。修后判断仍是不推倒、不重创，继续保留骨架。

## 2. 分段判断

### 001-010：能用，开局不需要重写

成立点：

- 001第一屏有公共灾难和可见羞辱：赵成被改成示材台，立刻证明“改名即改用途”。
- 卿禾的燃灯童、卿宸抢回一竖、无名船骨、七十人名额，都有清楚物理代价。
- 006-010用饭、锅、灶灰、半声到，把“活下去”和“拒绝白舟”落到了人和物上。
- 010结尾完成第一阶段选择：不是卿宸替全船决定，而是船民带着怕死留下。

轻修点：

- 001-004的概念连续进入太快：舟材世界、燃灯童、无名船骨、七十名额、拆名官、定名钉同时压进来，听众容易先被设定打中，后被人物追上。
- 006-010资产很密：归家灯、灶火舱、伙料、追踪香、同锅灰、半声到、童灯队债务都好，但需要一条更反复、更口语的明线兜住，比如“先让这船上的人吃一口饭，再去还童灯队的账”。

处理建议：

- 不改剧情顺序。
- 轻修001-004每章开头或结尾的1-2处，让“卿禾/七十人/童灯队”成为更稳定的听感锚。
- 006-010不用删资产，只要多用锅、饭、到、童灯队这几个朴素词压住术语。

### 011-020：不重写，但适合做一轮听感软修

成立点：

- 011苏晚出场很稳，“药归箱，人归人”是可复述句。
- 014铁牙出场有职业声音，“我认兵，不认料”成立。
- 018-019是前20章最关键的情绪节点：卿宸护妹过度，卿禾主动点灯，童灯队未救，旧校牌记忆损耗，逻辑和情绪都通。
- 020结尾结算明确：灶火舱、回声医舱、兵骨舱、登船索、童灯队债务都压进黑航道。

轻修点：

- 011-017连续开新功能位，听感可能从“妹妹/童灯队”滑到“医舱/兵骨/压舱者”。
- 压舱者线是好线，但夹在医舱、兵骨、童灯队之间，可能让听众短暂忘记“前20章到底追哪口气”。

处理建议：

- 不做大改。
- 以后回修时，在011、014、016、018的章首或章末轻轻回扣“童灯队债/卿禾灯耗/黑航道前路”，让它们像旧钟一样每隔几章响一下。
- 016-017只需确认压舱者线没有把节奏拖成伦理讨论；现有文本已有动作和保命书，可保留。

### 021-030：整体成立，不建议重构

成立点：

- 江城老街、市井舱、云织、许青刀、合法港口、拖城索、第一规互相咬合。
- 024合法港口的诱惑是有效的：水、药、床位、登记名有现实吸力。
- 027-030沈砚用假裂名定位钉反打未名舟规则，敌人聪明，不是只来送。
- 030第一规不是会议结果，而是从旧校牌、空碗、断秤、锅盖、绷带、钥匙、红泥残响里长出来，方向正确。

轻修点：

- 公议/船规/自愿陈述权这类规则词出现时，可以继续优先让木牌、铅笔、伤情牌、假裂钉来解释，少让人物说清定义。
- 024-030很密，但密得有战果；暂不需要动。

处理建议：

- 保留。
- 等第一卷后半段稳定后，再统一检查术语是否过密，不在当前阶段拆。

### 031-039：目前最贴近新版skill，优先保留

成立点：

- 031悬赏榜把敌方经济重新抬起来，水、药、床位、基础舱籍都变成诱饵。
- 032见裂样本被取走，敌人学会反制，后续假裂/假回应火有根。
- 033乡愁面失败是好过渡：不是水章，既有呼吸，也牵出回应火与假裂。
- 034-036试岸、三日清理令、隔水三息连续成立，普通人和旧物开始真正改变战局。
- 037-039听见者名单、巡听灯、半块饼、第二口债，追读钩强，章末小手非常适合直接接040。

轻修点：

- 031-039章节普遍变长，单章听感没问题，但后续要防止每章都做成小高潮。
- 040-050可以安排044-046一小段半日常，让读者喘气，同时修锅盖、补半饼槽、推进许青刀母子关系。

处理建议：

- 保留，不重写。
- 直接按`12_031-050阶段补纲_拒名船悬赏榜.md`续写040。

## 3. 是否符合最新版skill

整体符合。

符合项：

- 规则大多通过物件伤人或救人，而不是先讲定义。
- 章末基本都有物理残留：船骨、药箱、红鞋带、旧校牌、定位钉、红泥、回应火、半饼、小手。
- 普通人越来越能改变战局，尤其021以后明显变好。
- 敌方经济持续存在：合法港、悬赏榜、基础舱籍、儿童保护床位、听见者名单、巡听灯。
- 031以后听感显著升级，旧物、口语、行动链更稳。

不够贴的地方：

- 前20章偶尔像“资产连续解锁”，而不是“一条明线牵着资产长出来”。
- 有些章节规则名很强，人物情绪和旧物足够，但听书时仍可能需要更朴素的重复锚。
- 卿禾的旧校牌、糖水记忆、童灯队债务可以更早、更轻地反复碰一下，避免听众忘线。

## 4. 修订优先级

### P0：暂不建议现在重写

当前正文没有结构性断裂。若现在重创，反而会损失031-039已经形成的手感和旧账复利。

### P1：继续写完040-050

### P2：001-020已做入口轻修，后续只做随写随校

已处理方向：

1. 给001-010加一条更清楚的听感明线：救卿禾、让七十人吃饭、欠童灯队。
2. 给011-020加一条回扣：童灯队债和卿禾灯耗不要被医舱、兵骨、压舱者完全盖住。
3. 不删设定，只把正式名词后移半拍，让锅、药箱、旧校牌、鞋带先说话。

后续若再修，只处理读感出戏、入口不够抓、回声断线三类问题，不为规则表或字数做机械改动。

### P3：暂不动021-039

021-039已经是当前项目最稳的样板区。后续写作应该更多学习031-039的手感，而不是重写它。

当前已写章节不需要重构，更不需要重新创作。001-020已经完成一轮入口和回声加固；下一步应继续写完040-050，同时在后续创作里持续软校“读感不出戏、听感不断线”。
