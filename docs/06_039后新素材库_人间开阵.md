# 039 后新素材库：人间开阵

本文件是 039 后新增战场素材入口，只收新大纲能直接供正文调用的猎足、人间开阵、奇锋军团和后续阶段素材。它不替代 001-038 已经成立的正文事实；写 039 起必须先接住那些人、旧物、伤口和章末钩，再从这里生长新敌招、新战法和新代价。

## 1. 素材使用原则

1. 素材先服务战场，不服务解释。
2. 人先动，物才应阵。
3. 每个旧物必须有普通日子的原用途，也必须有被迫上战场后的失日债。
4. 敌人必须会学习，不能只当被打脸工具。
5. 每两章至少给读者一个小奖：敌招样本、救回影子、新战法、残舟回应、敌方误判或可研究碎片。
6. 039-050 只打第一只猎足，不解释整个宇宙。

## 2. 敌方素材：终相猎场

### 第一只猎足

外观不先写成怪物。先写成压迫：

- 黑水上没有水花，只有影子被踩扁。
- 甲板没裂，人却觉得自己的用途被压窄。
- 旧物没有碎，旧物和人的关系先被踩住。
- 脚印每落一次，船上就少一种选择。

三轮打法：

1. 踩影：确认人和旧物之间的关系。踩钥匙影，就知道谁还相信旧城有门；踩校牌影，就知道谁还想回学校。
2. 拴影：不抢人本体，先拖走他和旧物之间的影子。人还站着，却开始忘记自己为什么护着那件东西。
3. 换影：把旧物关系换到错误对象身上，让旧物误伤自己人，逼未名舟提前开阵。

可见战果：

- 被踩过的影子边缘变薄，像被压成一层黑纸。
- 残舟帆影被拖沉一寸，船体本身还没动。
- 旧门影子朝外开，现实里的门却没有动。
- 校牌照出的不是名字，而是一个被压成用途的轮廓。

可掉落样本：

- 足印碎黑：一小片像冻住的影子，能短暂显示敌人下一次要踩哪条关系。
- 反向鞋钉：钉尖不扎肉，扎“用途”；可作为后续沈砚压制开阵的样本。
- 空白脚泥：沾在甲板上，能把旧物短暂压回单一用途。

## 3. 战场素材：黑水猎足战

战场不只是甲板，要让读者觉得地球日常被拖进万界猎场。

- 黑水：像被拍卖场灯光照过的水，能映出旧物被标价后的影子。
- 三丈线：原本是残舟与未名舟互不吞并的距离，039 后变成伏击距离。
- 船头木牌：不是告示，是临时战场坐标，能写“可退、可错、可再答”。
- 旧绿门：现实里是市井舱入口，影子里可能变成敌方登船门。
- 残舟帆影：不亮灯残舟不并船，但帆影可以先并战。
- 甲板钉缝：猎足踩不到船骨时，会先踩钉缝里的旧名灰。

039-050 的场面递进：

- 039-040：看见猎足存在。
- 041-042：确认猎足踩的是关系。
- 043-044：普通人第一次把日常押上战场。
- 045-046：潜入敌后和救回影子。
- 047-048：敌人反适应，旧物差点误伤自己人。
- 049-050：缺口碗扣影成阵眼，人间第一次完整开阵。

## 5. 人物战位素材

卿宸：

- 不负责喊口号到处赢。
- 每章至少做一个不可替代动作：挡第一下终相、看出另一面、错开落点、递回选择权、压住猎足一息。
- 最帅的瞬间不是替别人开阵，而是把最疼的一息争回来，让别人自己站出来。

卿禾：

- 不是记忆代价容器。
- 她的主动选择是“我知道会忘，但这个人得先回家”。
- 039-050 至少一次照回错误影子，证明敌人聪明；再由她修正。

老周：

- 不当温情厨子，要当赤釜军的种子。
- 嘴上骂锅，手上把锅底烧穿。
- 锅烟遮猎足时，不写玄，要写呛、辣、焦、饭香和饿。

邱成：

- 不当心疼碗的笑点工具。
- 他最狠的动作是最渴时倒扣碗。
- 缺口碗扣住影子后，他失去“先喝一口”的资格。

梁春枝：

- 不只是分账。
- 她能称出“敌人以为没有重量的东西”：影子、迟到、欠命、旧约、残舟不愿并船的那一点距离。

孙福来：

- 不只是开锁。
- 他开的是敌人以为不存在的后门。
- 钥声被采样后，后续敌人能用假钥声诱他。

苏晚：

- 051 后重点，但 039-050 可先埋。
- 她不把伤者交给合法救援床，才有续脉奇锋。

铁牙：

- 不当纯武力。
- 他负责把“不能被推去当兵料的人”挡在身后。

残舟：

- 不并船，不归顺，不跪谢。
- 它的价值是证明未名舟不是唯一答案。
- 039-050 只让帆影先并战，船体保持距离。

## 6. 七奇锋与六大战军素材

七奇锋当前只露种子，不急着正式封号。

- 开阙：旧钥串、旧绿门、门缝、敌后门。
- 返灯：旧校牌、错字本、被拖走的人影。
- 悬衡：断秤、缺口碗、影子的重量。
- 续脉：旧绷带、药箱、病床、伤者本人回应。
- 断渡：焦索、船钉、旧桥、接舷。
- 影钩：买名市场、假卖假降、敌方欲望。
- 回昼：钟楼、迟到的车票、倒计时失效。

六大战军当前只露后勤和战场器官，不急着封军。

- 赤釜军：从锅烟遮猎足开始。
- 白垒军：从移动救援仓和旧绷带开始。
- 裂桥军：从接舷、拖船、断渡失败代价开始。
- 归轨军：从车站、铁轨、旧桥和撤离线开始。
- 晨灯军：从卿禾照回人影开始。
- 守岸军：从临时岸第一夜开始。

## 8. 051-100 预备素材

051-060 水药床战场：

- 合法救援港给真水、真药、真床，但要求登记用途。
- 水线、病床、药柜、旧绷带、伤者本人回应，组成移动救援仓。
- 续脉奇锋从“抢伤者一口气”出现。
- 白垒军从“能守住伤者的后勤器官”出现。

061-070 沈砚压制开阵：

- 沈砚不只定人名，而是定“地球旧物不得成兵”。
- 锅被压回锅，床被压回床，校牌被压回证件，钥匙被压回权限。
- 卿宸第一次开相压场，不替人决定，只把多个普通人的反用途动作接成战场。

071-080 双船并肩：

- 残舟有自己的船脸、旧物、脾气和退约权。
- 船团第一约是“不并名，只并力”。
- 敌人试图用共同船名诱导归并。

081-090 废名浅滩：

- 旧路牌、床号、门牌、柜门堆成浅滩。
- 合法打捞队把地球组件当垃圾分类。
- 未名舟第一次主动截押运线，变成敌方口中的盗名者。

091-100 临时岸：

- 学校门砖、菜市场青石、旧门、锅灰、绷带、断秤钉、校牌冷光共同缝岸。
- 岸能站一夜，但不能逃。
- 命名庭宣布非法陆基，白舰群压来。

## 9. 新素材生长口

后续需要水、床、药、锤、证物、学校、车站、医院、钟楼、市场或城墙时，直接按新大纲造：

- 先问它正在被高位面压成哪一种唯一用途。
- 再问哪个人不肯只剩这种用途。
- 然后让行动、战场、失日债和敌方反制一起出现。
- 最后沉淀成战法、船体器官、奇锋种子、军团器官或临时岸材料。

一句话：素材不是回收，是人间现场长出来的兵。
