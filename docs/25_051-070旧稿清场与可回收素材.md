# 051-070 旧稿清场与可回收素材

本文件记录一次创作口径清场：旧 051-070 已从活正文目录移除，不再作为连续续写事实，也不再要求后文承接旧节奏。

## 为什么清场

001-050 已经重构成新的书型：地球被拆成货，未名舟在黑水末世里抢人、抢水、抢药、抢旧物、抢航路、聚残舟。

旧 051-070 的标题和节奏容易把后文拉回“票据/钳子/规则解题”模式。它们有可用的物件，但不能继续当章节骨架。

## 后续写作口径

- 051 从 050 章末的“退约牌”重新发动。
- 敌人不再只是出题，而是用真水、真药、真床位诱钓残舟和船民。
- 每章先问：这一章他们会缺什么、抢什么、失去什么、打退什么。
- 命名权只能成为末世压迫和战斗机制，不能替代生存压力、登船战、追击战、夺物战。
- 主角卿宸要继续变强，但不能无痛硬解；他的抢笔、写距、写骨都要付出身体、旧物或关系代价。
- 配角不能只拿旧物站桩，要在压力里做选择：怕饿、怕渴、怕拖累别人、想换药、想守船、想把坏东西修到还能用。

## 可回收素材

这些旧标题和概念可以回收，但必须换发动机。

- 退约牌：保留。新 051 的核心钩。它不是规则题，而是干净、真实、能救命的水药诱惑。
- 夜航旧票：可回收为黑航道夜间通行物。重点不是验票，而是夜追、错航、白灯逼近、残舟是否掉队。
- 白灯夜航：可回收为敌方夜间追击战。白灯要照出船损、病人喘声、锅汽和残舟影子。
- 退票口：可回收为临时岸/退路陷阱。不要写成窗口规则，要写成“只要过这个口，就能上岸睡一觉，但会被写成合法离船者”。
- 补票钳、剪票钳影：可回收为近身登船武器。钳子要咬索、咬门、咬手腕、咬船骨，不要只咬概念。
- 掌舵手影：可回收为卿宸主角高光。不是影子谜题，而是一次黑航道大偏航，只有他能看裂抢舵。
- 不像饭 / 食者先坐：可回收为粮食线。写饥饿、假饭、可食用登记、分饭争执和一口真饭的代价。
- 船板亦可量 / 船板待钉：可回收为修船线。写船真的要裂、要沉，修船钉真实有用但会把船写成命名庭资产。
- 取船骨旧名 / 下问醒船人：可回收为船骨升级线。不能写成问答，要写成船骨被敌人夺名、卿宸以血写骨抢回一次偏航能力。
- 到者可采 / 缺席者可补点：可回收为童灯队、半声到、失踪者点名线。必须服务救人和战斗，不做名单游戏。
- 白线追声：可回收为追声灯升级。声音会暴露病人、孩子、残舟和灶火，迫使船队静音夜航。
- 冷岸不收：可回收为第一次临时岸失败。岸不是规则名词，而是湿冷、硌脚、能站人也会吃人的地形。
- 洗碗先问碗：可回收为邱成/缺口碗的人味小章或战后喘息。让碗参与活命，不写成哲理。
- 偷暖不登源：可回收为残舟互借热源。重点是偷一口暖气给孩子，不是讨论源头归属。
- 我想吃饭不想当灶：强保留。适合做 060 左右的小爆点：被写成临灶的孩子第一次完整说出自己的欲望。

## 051 新发动机提醒

050 章末状态：

- 同行三十丈 / 可退 / 不并船刚成立。
- 不亮灯残舟跟着走，但没有敲。
- 第二道回应火亮了一息，还未靠近。
- 清水一袋和退热药半瓶仍没拿到。
- 退约牌写着：退约即给水 / 单船可领 / 先到先得。

051 应写成诱钓和生存压力章：

- 水要真实有用，最好能让发烧孩子马上好一点。
- 残舟或船民动摇不能是蠢，而是“不想拖累别人”“孩子要渴死了”“药真的在眼前”。
- 未名舟不能只拒绝，必须抢到一点更难的活路，比如从退约牌边缘刮下湿气、逼出一口不归属的水、或让残舟自己选择退半丈再敲回来。
- 章末要把敌方从“诱钓”推进到“夜航追击”或“白灯逼船”，给 052 留更大的动作场。
