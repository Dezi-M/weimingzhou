# 《未名舟》十卷长线与001-200阶段路线

本文件是当前长线与 001-200 阶段路线，负责把新大纲压成可连续推进的章段主菜。

## 1. 总原则

两千章不能重复“某人被改名，主角抢回名字”。每一卷尽量更换被命名对象、更换敌方逻辑、更换船体资产、更换爽点形态。

宏观升级：

单船活命 -> 船团救人 -> 舰队开战 -> 拼出陆地 -> 万界自命。

本书的宏大感不是战舰数量，而是“普通人的选择和地球组件能托起文明”。每一卷的大场面都要落回饭、药、灯、门、木牌、旧街和普通人高光。

每卷还尽量有一个反终名脑洞：敌人给出一个看似更稳定、更正确、更能救人的唯一答案，未名舟证明人和世界不能被这个答案封死。

| 第二卷 | 合法航道/港口 | 没有合法名的船能不能抵达明天 |
| 第三卷 | 世袭职业 | 一个人明天还能不能换活法 |

| 第六卷 | 神明能源/旧信仰 | 神明明天还能不能不是电池 |

每卷主菜可以很燃、很战斗，但深处都要咬住这一口：谁在卖明天，谁在抢回明天。

## 4A. 大纲脑洞审计闸门

每次做卷纲、十章纲或重排路线，先不要急着顺旧表往下排。先用六个问题审大纲：

1. 这十章有没有一个读者没见过的高概念奇观，并且能落到饭、药、船、门、灯、伤口或旧物上？
2. 敌人有没有主动升级打法，而不是等未名舟来破解？
3. 卿宸有没有一个只有他能做、但不能替别人决定的动作？
4. 普通人、旧物、奇锋、军团、残舟、未名岸之间有没有形成新组合，而不是各自出场？
5. 赢了以后有没有反噬：敌方新样本、失日债、外债、伤口、误会、资源缺口或更大围剿？
6. 这十章读者能不能用一句话记住：抢回了地球哪一块，它变成了什么战力？

若六问里三问以上答不响，大纲不是细节不足，而是主菜不够，需要先头脑风暴再重排。

头脑风暴允许越界：

- 可以直接重排当前章段。
- 可以新增敌方打法、外界种族、买家势力、战场地形、特种小队用法和军团封军战。
- 可以让地球组件、旧职业、旧城市以更爆裂的方式上战场。
- 可以让敌人短期赢得漂亮，逼未名舟用更聪明、更痛的方式赢回来。

落笔必须回航线：

> 再大的脑洞，进入正文时都要压成一个当场危险、一个具体动作、一个可见收益和一个章末钩。

## 5. 第一卷001-200总目标

第一卷目标不是普通“逃出拆解区”。

> 抢回地球第一枚自命印，拼出第一块不被命名庭承认也能站住的岸。

读者短线追：妹妹还能不能救、七十人怎么活、童灯旧债会不会反咬。
读者中线追：下一块地球组件会被抢回来什么，它会让未名舟多出什么战斗力。
读者长线追：地球不是复原成昨天，而是拿回自己继续变化的明天。

新版第一卷定调：

> 从 001 开始，第一卷就要让地球第一次把人间日常打成战场。

旧版 001-038 已清理。新版从 001 就把“终相猎场 vs 人间万象”埋进活拆地球、第一口饭、急救方舱、旧城上船和船外残舟。

## 7. 039-050 新引擎：黑水猎足战与首次人间开阵

039-050 是新版反围猎战争的第一场爆点。

038 章末已有无形脚印。039 不要立刻讲大宇宙，只让脚印踩影：踩残舟帆影、钥匙影、缺口碗影、旧校牌影。读者先感到恐怖，再慢慢知道这是终相猎场伸进黑水的第一只猎足。

上帝视角补钉：

- 这十二章不能只有“猎足越来越近”。每两章给读者一个小奖：看清敌招、救回一个影子、试出一个旧物战法、逼残舟主动回应、让敌人留下一块可研究的足印碎黑。
- 卿宸不能只是等 050 喊名场面。他从 039 起就要持续付身体代价，挡第一下、错开落点、拆穿敌人的第二层陷阱，并把选择权递给普通人。
- 猎足不是无脑怪。它有三轮聪明打法：先踩影确认关系，再拴影拖走人和旧物，最后换影让旧物误伤自己人。
- 050 不是突然开挂，而是前十一章所有小失败、小胜利、小代价汇到一起，终于让读者觉得这一句该喊。

039-050 执行约束：

1. 不让文书、税名、榜单当主菜；它们只能做敌方压迫的表面。
2. 敌人聪明：它不砍旧物，先踩人和旧物之间的影子关系。
3. 每章至少有一个人先做选择，再让地球组件回应；不能只写旧物显灵。
4. 先小胜，再反噬，再由普通人选择成阵。不要一句“人间开阵”直接赢。
5. 050 前必须结出看得见的大东西：第一只猎足被打断、第一套万象开阵成立、残舟不并船但并战。

039-050 推荐小节奏：

- 039：猎足踩影，大家误以为只是普通登船压迫。
- 040：锅灰描足，旧门差点变成敌方入口。
- 041：猎足踩残舟帆影，不亮灯残舟被拖沉一寸。
- 042：三丈线从关系约定变成伏击线。
- 043：老周锅烟遮猎足定位，锅底烧裂。
- 044：梁春枝称影，断秤崩角。
- 045：孙福来开敌后门，钥匙断半串。
- 046：卿禾照回人影，失去一段普通校园记忆。
- 047：残舟第一次主动开火，但仍不并船。
- 048：敌人布下反开阵陷阱，逼旧物错用。
- 049：邱成倒扣缺口碗，不接水，扣影成阵眼。
- 050：卿宸第一次完整喊出“人间，开阵”，打断猎足，残舟同行三十丈。

039-050 奖励曲线：

- 039-040：读者得到恐怖感和第一个敌招样本，知道“脚印踩的不是甲板，是关系影子”。
- 041-042：读者得到第一场埋伏，三丈线不再是约定，而是战场距离。
- 043-044：读者得到第一组普通人高光，锅烟和断秤不是显灵，是老周、梁春枝主动把日子押上去。
- 045-046：读者得到潜入和救回，钥匙开后门，校牌照回影子，但两件旧物都伤到回不去。
- 047-048：读者得到敌人反适应，残舟主动出手，却被敌人拿旧物关系反设陷阱。
- 049-050：读者得到阶段大爽，缺口碗扣影成阵眼，卿宸用前十一章攒出的战位开阵，只打断第一只猎足，不解决整个宇宙。

## 8. 051-200后续阶段

### 051-060：水药床战场与移动救援仓

猎足战后，敌人不再只验桥、收税、追票，而是试图把水、药、床重新压成“救援诱降”。合法港给真水、真药、真床，但要求登记用途和交出开阵样本。未名舟不能只拒绝，要主动抢一座移动救援仓：水线、病床、药柜和旧绷带共同成为续脉奇锋种子，也成为白垒军的后勤根基。结算：续脉第一次从药源名里抢人，白垒军第一次有了能守住伤者的战场器官。

### 061-070：沈砚压制人间开阵

沈砚正式意识到：地球组件不是材料，而是人间开阵的战场器官。他不再只定人名，而是试图定住“地球组件不得成兵”。饭火、病床、校牌、门路、衡量和救援都被压回单一用途，卿宸必须从抢笔升级为开相压场。结算：沈砚获得针对人间开阵的第一份样本，卿宸第一次让多个文明器官并相反击。

### 071-080：双船并肩战与临时船团第一约

不亮灯残舟和贴票残舟都要有自己的脸、旧物、脾气和不愿意。未名舟不收编它们，只挂约同行，打一场错位小船团战。结算：临时船团第一约成立，不并船、不取共同名、各自可退。

### 081-090：废名浅滩与第一次主动资源战

资源压迫重新顶上台面：水只剩湿底，粮只够两顿，药布不够，船必须抢。废名浅滩由旧路牌、床号、门牌、柜门堆成。合法港打捞队和命名庭押运线出现，逼未名舟成为“盗名者”。结算：半柜浑水、过期药、学校门砖、菜市场青石和未名岸材料入手，同时留下第一枚买家货标碎片。

### 091-100：第一块临时岸站住一夜

黑航道起字风，船不能一直逃。云织用旧门、菜市场青石、锅灰、绷带、断秤钉、钥匙锈、校牌冷光和错题纸角缝第一块临时岸。结算：锅能放稳，伤者能躺平，命名庭宣布非法陆基，白舰群压来。

### 101-120：临时岸守夜战

第一块岸不是安全屋，而是要守的家。命名庭和合法救援舰群用水、药、床位和合法地名诱惑岸上人。结算：岸防雏形、守线桩、药柜位置、第一张课桌或临时睡铺成立。

### 121-140：买家押运线与地球组件外流

未名舟第一次真正追到买家航线。江城钟楼声、急救地板、学校门砖或童灯旧名被送往别的世界。买家不一定脸谱化，他们有自己的灾、饥荒、战争或秩序需求。结算：抢回一批被装进货单的人和旧物，留下第一枚买家标记或叛逃者线索。

### 141-160：承约墙雏形与第一笔外债追偿

求救木牌升级为临时承约墙，但不是任务面板，也不是道德审判台，而是处理分债、退约、不归属和追偿的战斗设施。结算：承约墙能接活，也能记债、挡伪约、反追买家链。

### 161-180：地球自命印开门

地球第一枚自命印线索明确，但打开它不靠单一钥匙，而需要城市旧名、孩子旧声、船民选择、敌方钉证、外债见证和卿禾自己的判断。结算：自命印红泥显出“复名世界”的本质。

### 181-200：第一卷大结算

未名舟夺回地球第一枚自命印，用自命印红泥、江城旧物、挂约残舟和一百块求救木牌稳住临时岸。但童灯队三个孩子没能全部救回，其中一个被沈砚改造成反未名舟武器。卷末大爽带大痛：正义赢大局，留下复仇、船团、外债、未名岸和第二卷废名港航线。

- 080：第一场双船并肩战，临时船团雏形。
- 090：第一次主动资源战，水药粮和未名岸材料。
- 100：第一块临时岸站住一夜，非法陆基宣告。
- 150：承约墙雏形成形，敌方合法承约厅第一次挖人。
- 200：地球第一枚自命印结算，临时岸与船团进入第二卷。

每十章读者都要能说：这十章抢回了地球哪一块，它怎么变成战力，人强了什么，敌人学会了什么，下一段我想看他们再劫回什么。

## 10. 跨卷关系复利

| 关系 | 起点 | 中段变质 | 长线复利 |
|---|---|---|---|
| 卿宸 / 卿禾 | 保护与被保护 | 卿宸学会放手，卿禾主动点灯 | 并肩决定地球如何归来 |
| 卿宸 / 陆小满 | 质问与被质问 | 共同制定可战斗的规则 | 陆小满成为能公开限制强者的追问席 |
| 卿宸 / 沈砚 | 拒名者与拆名官 | 两种救人哲学正面对撞 | 打败沈砚不是杀死他，而是打穿“错误名字也比无名强” |
| 老周 / 船民 | 做饭的人与挨饿的人 | 灶火舱成为士气中心 | 乡愁面成为归船、婚礼、送别和战后庆功符号 |
| 苏晚 / 病人 | 医者与伤者 | 救命债公开化 | 回声医舱成为“不卖病人”的制度资产 |
| 铁牙 / 守线队 | 老兵与新兵 | 不把弱者推到身后 | 兵骨舱从战争机器变成守线军团 |
| 云织 / 江城 | 缝补者与残城 | 找一扇缝不回去的门 | 市井舱成为未名舟最能留人的烟火核心 |
| 迟灯 / 承约墙 | 骗子与规则 | 假背叛测试信任 | 买名市场线的长期反转发动机 |

关系复利要求：每次大战后至少有一组关系留下新称呼、新物件、新误会或新承诺。
