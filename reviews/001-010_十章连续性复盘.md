# 001-010 十章连续性复盘

## 十章总判断

复盘范围：001《成交以后》到 010《拒名船身份》。

整体链路成立：第一小段已经把书型卖出来了。地球被合法拆卖，卿宸不是辩论而是违法抢人，未名舟不是天降神器而是人和旧物互相拽出来的活路。十章结算到“破船能活人”：能抢一口水、一点药、一口饭、一条饭线，也能拒掉敌方恶名。

最强爆点：010 七十个不一样的口印咬裂“饥舟”白牌。

相对弱项：006-010 连续高压偏长，听书会有一口气绷到底的感觉；011 可以继续高压，但需要用床轮、药味、苏晚旧绷带给读者一个新的行动入口，而不是继续空碗/白牌同类压力。

最大未结旧债：苏晚和水底抢救室仍未真正救回；011 已自然接到急救方舱，是合适的回收口。

## 资产与代价清单

- 船体资产：无名船骨、第一块歪甲板、白泡缝路、灰白医线、伤铺边、非法护手钩、非法饭线、七十道咬痕、半块白牌。
- 生活资产：半饭盒盐水、一角止血棉、第一口不登记热粥、锅边饭味。
- 见证资产：成交金线、本人不退价牌、不代、锅边、嘴有、七十个不一样的口印。
- 人物代价：卿宸掌心和虎口多次受伤，卿禾记忆缺口增多，陈一桥腿伤加重，梁朵饭盒反复裂，周安铅笔消耗殆尽，罗琴失工牌且腿伤未结，孟达仍困担架。
- 敌方学习：非法夺货、盗舟、未结案伤口、非法急救、代登失败、陪护项回收受阻、自取进食、非法饭线、统一口印失败、未名舟记录暴露。

## 旧账折痕

- 苏晚旧绷带：还在灰白医线里颤，011 抢床时可以让她的“别停”成为医线核心，不急着本人上船。
- 卿禾记忆缺口：已经忘过周安和梁朵的部分信息，后续点灯时可以轻轻碰，不要只当能力消耗。
- 周安新铅笔：铅芯碎、木杆断，欠一根带橡皮的新铅笔，这个小愿望很适合以后低压时还。
- 梁朵新饭盒：饭盒承担太多，裂口越来越危险，后续抢旧城 / 市井舱时可以给她一个“不漏”的回收。
- 七十口气：010 只是口气先挂船，不是七十人全治好；后续不能忘记车厢伤者身体还在急救方舱压力下。

## 敌方经济小记

- 亲属优先赎回：用亲情换交还，第一轮交易诱饵。
- 登记后可领用：真水真药真床，换伤者登记。
- 陪护项 / 陪护餐：真热粥，换陪护人和喂食者身份。
- 七十口饥餐：真饭，换“饥舟”临时船名和统一口印。
- 急救方舱：真药真床真命，换“伤者”用途登记。

敌方经济压力稳定，而且越来越聪明：从买卖人和物，升级到拿真实资源换身份确认。

## 连续性问题

- 硬断裂：未发现。
- 软弱项：006-010 都围绕急救车厢 / 白柜 / 腕带压力，连续性强但场景材质接近；011 需要换成床轮、药柜、方舱空间感，避免继续像同一个白柜副本。
- 待回收：罗琴旧工牌、孟达身体脱困、苏晚本人、七十人身体上船、卿禾记忆缺口。

## 名场面与爆点

- 001 卿宸从“成交”二字之间扯下一寸金线。
- 002 卿宸在船骨上刻“欠”，救人从标价变成欠命。
- 003 梁朵按住三粒泡白米说“我饭还没吃完”。
- 006 罗琴说“别替我登”。
- 007 梁朵说“那我替我自己”，饭盒裂口咬出护手钩。
- 008 五个人各自自取第一口不登记热粥。
- 009 周安写下“嘴有”，七十张嘴让空碗数不过来。
- 010 七十个不一样的口印咬裂“饥舟”白牌。

## 角色主动欲望变化

- 卿宸：从救妹妹，变成不代答、不独占船名、把选择权递回每个人。
- 卿禾：从被选为燃灯童，变成主动判断照谁、照哪一面；她照人，不照敌方牌面。
- 梁朵：从怕被退货的小孩，变成护饭盒、护饭线、自己咬住旧绷带的人。
- 周安：从用铅笔答到，到没笔芯也用木杆卡白牌；他的欲望很简单：我还能写一点、卡一点。
- 罗琴：从被押运的未登记伤者，变成拒绝代登、提醒别人“这回也别替我”的成年人。
- 孟达：从车厢里的远处伤者，变成用外卖旧业证明自己不只剩饥饿的人。
- 老周：从锅灰里浮出来的人，变成赤釜线早期核心，嘴硬但一直把饭、灰、热味压到能救人的地方。

## 节奏与读者情绪

001-003 是成交灾难和盗舟逃生；004-006 是急救伤口和登记陷阱；007-010 是饭、嘴、饭线和拒名身份。每三章有升级，十章小高潮有结算。

读者情绪链：震惊地球被卖 -> 跟卿宸抢人 -> 看见破船能救人 -> 被真水真饭诱惑 -> 爽在众人自己答、自己取、自己咬下恶名。

下一段 011-020 应该把“资源战”从饭线切到医药床和第一场战斗。建议让 011 开场就有床轮、药味、二次救治牵引，不要从复盘 010 开始。

## 下一步建议

- 011 从“先抢床”直接开：病床轮压近、真药味出来、伤者被标成用途。
- 苏晚旧绷带要动，不一定救出苏晚本人，但要让“别停”参与抢床。
- 抢床不是抢福利，而是抢一个可以守伤者的临时器官。
- 给急救方舱一个聪明反制：它不怕未名舟拒绝，它怕伤者自己拒绝“只当伤者”，所以会给止痛和安稳诱惑。
- 011 结尾最好落在一个新的医线 / 床栏 / 药柜碎片，而不是又一块白牌。
