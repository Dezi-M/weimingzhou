# 041 正文审计：拒灯船也要吃饭

时间：2026-06-22

正文文件：`chapters_txt/第一卷_地球舟材/041_第四十一章_拒灯船也要吃饭.txt`

## 总判断

可发。本章没有继续堆新规则敌人，而是把 040 的“拒灯船群”爽点压回末世生存账：锅盖漏、伤者要热水、外船也冷也饿，命名庭递来真米、真钉、真布，逼未名舟承认“主供船”。章节功能是从宏大钩子落到吃饭和修补，非常必要。

## 八格审计

- 危险：拒灯船群刚出现，未名舟自己缺粮缺热，外船也要热；白箱以救济粮和修补钉诱导确认主供船。
- 目标：借一口热给外船，但不领取群粮、不定主船、不让三条水路并成主船。
- 敌压：拒灯船群临时救济箱，干米、修船钉、止血布、火绒都是真的；条件是确认主供船并负责分配、登记、归并。
- 行动：老周开锅借热，第三船刮烧焦桨木芯，孙福来用钥匙偏水缝，陆小满写“各出旧物 / 共借一口热 / 不领群粮 / 不定主船”，卿宸用青烟熏散主船痕。
- 战利品：一层临时热雾，三船各自喘到一口热气；主船未定；群粮未领。
- 损失/代价：没拿米、钉、布、火绒；第三船烧焦桨耗掉一点木芯；卿宸掌心被青烟烫泡；旧物破损继续无人代修。
- 感官：锅盖漏汽、干米香、冷水风、焦木青烟、孩子吸鼻子、远处钉匣敲铁声。
- 关系：老周嘴硬但借热；邱成仍心疼碗但没有拖后腿；云织失线后用手腕认断；孙福来从钥影危机后继续守“不开人”；外船有了“也冷也要热”的活人欲望。
- 宏大线：拒灯船群不只是船影数量扩张，而是带来供给、修补、主船责任和招安诱惑。

## 意外钩与情绪钩

- 获得什么：临时热雾、各船“各出旧物”的互助方式。
- 表面用处：让外船和未名舟都喘一口热气。
- 真正用处：建立拒灯船之间“不并船、不定主、各认各破”的船群协作原则。
- 代价：拒绝真实粮钉布，改征修补税；旧物损耗继续累积。
- 旧账折痕：群粮箱未领不是胜利爽完就算，它的米香、修船钉和止血布都真实有用，后面还会诱惑船民。
- 情绪链：船群燃点后的冷与饿 -> 米香诱惑 -> 主船痕压迫 -> 借热小胜 -> 吃热水的苦笑 -> 钉匣来临的紧。
