# 041-050 十章复核：残舟保卫战

## 十章总判断

041-050 链条成立。它从“拒灯船也要吃饭”的资源压力出发，逐步把主船、主修、主源、刮除源、接客、同犯、旧物影、同行主体这些陷阱打成一条连续保卫战，最后落到“同行三十丈 / 可退 / 不并船”。最强爆点在 050，大孩子说“在 / 还 / 走”，残舟不归并却同行。最大旧账是水药床位仍没真正解决，下一章退约牌正好继续钓这个缺口。

## 资产与代价清单

- 资产：丑补法、分源反制、挡耳盖、刀背震耳、旧磨石问刃一次、外舷挂齿、裂扣、同欠账、白榜残网一角、白尺皮一截、同行三十丈试行。
- 船团种子：不亮灯残舟从远处钩子变成隔水同行者，第二道回应火稳住一息。
- 代价：群粮未领，清水/退热药/床位仍未拿到；锅盖、缺口碗、断秤、钥匙、旧绿门、焦索持续破损；卿宸掌心连续受伤；卿禾归家灯更薄；旧磨石问刃已耗。
- 敌方学习：主供、主修、主源、近源、刮除源、客钩、同犯、旧物影、距差/疼差，敌人一直在学未名舟的旧物用法。

## 旧账折痕

- 水药床位：一直真实有效但没领，051 退约牌正好继续压。
- 小白耳：从锅盖到磨石，仍在偷学，不急着清掉。
- 大孩子：已说“在 / 还 / 走”，但白蜡未全解，半饼槽仍欠。
- 第二道回应火：只稳住一息，后面要让它有自己的脾气和代价。

## 敌方经济小记

敌方经济非常清楚：群粮箱、钉匣修补、听源白税、清水袋、药包、床位、白榜影账、退约牌。它们都不是空设定，能真实改善生存，所以拒绝才疼。

## 连续性问题

无硬断裂。软弱项是 048-050 连续高压且都围绕水药床诱惑，读者会累；但 050 用大孩子恢复三个字和残舟同行完成阶段释放，暂时压住了疲劳。051 建议别继续同一种硬压，改成“钓”和“内部动摇”。

## 名场面与爆点

- 老周误喊锅盖属主后改口“破盖 / 漏盖 / 挡耳盖”。
- 许陈氏问磨石“钝在哪儿”。
- 孩子娘说“床我想要，可扣子不能替我娃睡”。
- 大孩子说“在 / 还 / 走”。
- “同行三十丈 / 可退 / 不并船”落到白尺皮旁边。

## 角色主动欲望变化

- 卿宸：从连续抢笔硬扛，转成分力掌舵和判断距离。
- 卿禾：灯越来越薄，但仍主动照低处，不抢主光。
- 老周：不只护锅，开始护船群的热气和骂声。
- 孙福来：从怕钥匙被当钥匙，到主动用钥齿卡住边界。
- 梁春枝：用市井损话称水、称疼、称债，越来越像船上的民间法度。
- 大孩子娘：从想要床，到亲手把袖线系在挂齿根上。

## 节奏与读者情绪

这十章总体是连续战斗，压力密。好处是追读钩强，坏处是水药床诱惑反复出现。050 结算给了“看得见的大东西”，下一段适合先用退约牌制造更细的人心裂缝，再安排一次主动抢资源，别只被动挡文书。

## 下一步建议

1. 051 用退约牌钓动摇，不让任何人降智，只让渴、病、怕拖累别人变成真实选择。
2. 第二道回应火要尽快拥有独特旧物或声音，避免只是“不亮灯残舟 2 号”。
3. 让卿宸继续减少伤手硬压，多用“分力掌舵”的主角魅力。
4. 052-055 可考虑安排一次小资源抢夺，不要只拒绝合法水。
